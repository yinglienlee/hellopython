邏輯與比較運算符號

學習導覽

邏輯與比較運算符號是 Python 中不可或缺的工具，用於處理條件判斷和邏輯推理。在本章中，我們將學習兩大類運算符號及其應用：

比較運算符號

  • ==：檢查兩個值是否相等。
  • !=：檢查兩個值是否不相等。
  • <：檢查左邊的值是否小於右邊的值。
  • >：檢查左邊的值是否大於右邊的值。
  • <=：檢查左邊的值是否小於或等於右邊的值。
  • >=：檢查左邊的值是否大於或等於右邊的值。

邏輯運算符號

  • and：當所有條件都為真時，結果為真。
  • or：只要有一個條件為真，結果即為真。
  • not：反轉條件的真值。

通過這些運算符號，你可以輕鬆地構建複雜的邏輯條件。

比較運算符號詳解

1. == 相等運算符號

== 用於檢查兩個值是否相等：

a = 5
b = 5
print(a == b)  # 輸出 True

c = 5
d = '5'
print(c == d)  # 輸出 False，因為類型不同

需要注意的是，== 會比較值的相等性，而不是記憶體地址。

2. != 不等運算符號

!= 用於檢查兩個值是否不相等：

a = 5
b = 10
print(a != b)  # 輸出 True

c = 5
d = 5
print(c != d)  # 輸出 False

當兩個值不相等時，結果為 True。

3. < 小於運算符號

< 用於檢查左邊的值是否小於右邊的值：

a = 5
b = 10
print(a < b)  # 輸出 True

c = 10
d = 5
print(c < d)  # 輸出 False

當左邊的值小於右邊的值時，結果為 True。

4. > 大於運算符號

> 用於檢查左邊的值是否大於右邊的值：

a = 10
b = 5
print(a > b)  # 輸出 True

c = 5
d = 10
print(c > d)  # 輸出 False

當左邊的值大於右邊的值時，結果為 True。

5. <= 小於或等於運算符號

<= 用於檢查左邊的值是否小於或等於右邊的值：

a = 5
b = 10
print(a <= b)  # 輸出 True

c = 5
d = 5
print(c <= d)  # 輸出 True

e = 10
f = 5
print(e <= f)  # 輸出 False

當左邊的值小於或等於右邊的值時，結果為 True。

6. >= 大於或等於運算符號

>= 用於檢查左邊的值是否大於或等於右邊的值：

a = 10
b = 5
print(a >= b)  # 輸出 True

c = 5
d = 5
print(c >= d)  # 輸出 True

e = 5
f = 10
print(e >= f)  # 輸出 False

當左邊的值大於或等於右邊的值時，結果為 True。

7. is 身份運算符號

is 用於檢查兩個變數是否引用同一個物件（即記憶體地址是否相同）：

a = [1, 2, 3]
b = [1, 2, 3]
c = a
print(a is b)  # 輸出 False，因為a和b引用不同的列表物件
print(a is c)  # 輸出 True，因為a和c引用相同的列表物件

is 與 == 不同，is 比較的是對象的身份（記憶體地址），而 == 比較的是值的相等性。

8. is not 身份運算符號

is not 用於檢查兩個變數是否不引用同一個物件：

a = [1, 2, 3]
b = [1, 2, 3]
c = a
print(a is not b)  # 輸出 True，因為a和b引用不同的列表物件
print(a is not c)  # 輸出 False，因為a和c引用相同的列表物件

當兩個變數不引用同一個物件時，結果為 True。

9. 特殊情況：連續比較

Python 支援連續比較，使條件判斷更加簡潔：

a = 5
print(1 < a < 10)  # 輸出 True，等價於 1 < a and a < 10

x = 20
print(10 <= x <= 30)  # 輸出 True，等價於 10 <= x and x <= 30

這種語法在檢查一個值是否在某個範圍內時特別有用。

邏輯運算符號詳解

1. and 運算符號

and 用於檢查多個條件是否同時成立：

a = True
b = False
print(a and b)  # 輸出 False

只有當 a 和 b 都為 True 時，結果才為 True。

2. or 運算符號

or 用於檢查是否至少有一個條件成立：

a = True
b = False
print(a or b)  # 輸出 True

只要有一個條件為 True，結果就為 True。

3. not 運算符號

not 用於反轉條件的真值：

a = True
print(not a)  # 輸出 False

如果 a 為 True，那麼 not a 就為 False，反之亦然。

綜合運用邏輯與比較運算符號

在實際撰寫程式過程中，我們經常需要結合邏輯與比較運算符號來處理複雜的條件判斷：

# 檢查一個數字是否在某個範圍內
number = 25
is_in_range = 20 <= number <= 30
print(is_in_range)  # 輸出 True

# 檢查年齡是否符合青少年條件（13-19歲）
age = 15
is_teenager = 13 <= age <= 19
print(is_teenager)  # 輸出 True

# 複雜條件：檢查一個人是否符合某項活動的參加資格
age = 25
has_membership = True
lives_locally = False
can_participate = age >= 18 and has_membership and (lives_locally or age < 30)
print(can_participate)  # 輸出 True

組合使用這些運算符號可以幫助我們構建精確的條件判斷。

常見問題與解答

問題 1：如何組合多個邏輯運算符號？

可以直接組合使用，例如：

a = True
b = False
c = True
print(a and (b or c))  # 輸出 True

問題 2：邏輯運算符號的運算順序是什麼？

Python 的邏輯運算符號遵循以下優先順序：not > and > or。

如果不確定順序，可以使用括號提高可讀性和控制運算順序。

問題 3：邏輯運算符號是否只能用於布林值？

不一定，Python 中的邏輯運算符號也適用於非布林值。例如：

print(0 or 42)  # 輸出 42
print(0 and 42)  # 輸出 0

Python 會根據 "真值" 的概念返回第一個能決定結果的值。

問題 4：True 與 False 以數值表示是什麼？

在 Python 中，True 等價於數值 1，而 False 等價於數值 0。它們可以直接參與數值運算。

print(True == 1)   # 輸出 True
print(False == 0)  # 輸出 True
print(True + 1)    # 輸出 2
print(False + 1)   # 輸出 1

問題 5：數值如何判斷為 True 或 False？

在 Python 中，非零數值（如整數或浮點數）會被視為 True，而零值（如 0 或 0.0）會被視為 False。

print(bool(42))    # 輸出 True
print(bool(0))     # 輸出 False
print(bool(-3.14)) # 輸出 True

問題 6：字串是否會被視為 True 或 False？

非空字串（即字串中包含文字或空白字元）會被視為 True，而空字串（""）會被視為 False。

print(bool("Hello"))   # 輸出 True
print(bool(""))        # 輸出 False
print(bool(" "))       # 輸出 True

問題 7：物件如何判斷為 True 或 False？

非空物件（如列表、字典、集合）會被視為 True，而空物件（如空列表 [] 或空字典 {}）與特殊值 None 會被視為 False。

print(bool([1, 2, 3]))   # 輸出 True
print(bool([]))          # 輸出 False
print(bool(None))        # 輸出 False

問題 8：== 與 is 有什麼區別？

== 比較的是值是否相等，而 is 比較的是對象的身份（記憶體地址）是否相同：

a = [1, 2, 3]
b = [1, 2, 3]
print(a == b)  # 輸出 True，因為兩個列表的值相等
print(a is b)  # 輸出 False，因為它們是不同的對象

# 常見的陷阱：比較None時應該使用is而不是==
value = None
print(value is None)  # 推薦的寫法
print(value == None)  # 不推薦的寫法

在比較 None 時，建議使用 is 而不是 ==。

問題 9：比較運算符的優先順序是什麼？

在 Python 中，比較運算符的優先順序高於邏輯運算符。完整的優先順序如下：

  1. 算術運算符（如 +、-、*、/）
  2. 比較運算符（如 ==、!=、<、>）
  3. 邏輯運算符（not、and、or）
# 例子
print(2 + 3 > 1 * 4 and 10 / 2 >= 5)  # 計算過程：(2+3) > (1*4) and (10/2) >= 5
                                       # 即：5 > 4 and 5 >= 5
                                       # 即：True and True
                                       # 結果：True

如果不確定運算優先順序，可以使用括號來明確指定。

# 解答

if (10 <= number <= 20) or number < 0:
    print("該數字在範圍內或為負數")
else:
    print("條件不成立")
	
# 另解
number = int(input("請輸入一個數字："))
if (10 <= number and number <= 20) or number < 0:
    print("該數字在範圍內或為負數")
else:
    print("條件不成立")

挑戰 8：複雜條件判斷 🛈

設計程式，模擬一個電影院的門票系統。兒童（12歲以下）和老人（65歲以上）享有50%折扣，學生（持有學生證）享有30%折扣，其他人則需支付全額票價。

# 解答
full_price = 300
age = int(input("請輸入年齡："))
is_student = input("是否持有學生證？(是/否)") == "是"

if age < 12 or age >= 65:
    price = full_price * 0.5
    print(f"您享有50%折扣，票價為 {price} 元")
elif is_student:
    price = full_price * 0.7
    print(f"您享有30%折扣，票價為 {price} 元")
else:
    price = full_price
    print(f"您需支付全額票價 {price} 元")

結語

邏輯與比較運算符號是程式中處理條件判斷的核心工具。掌握各種比較運算（==、!=、<、>、<=、>=、is、is not）和邏輯運算（and、or、not）的用法，你可以處理從簡單到複雜的邏輯需求。
