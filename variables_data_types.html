變數與資料型態

學習導覽

本章節將帶你認識 Python 中的變數與資料型態，並學會以下內容：

  • 了解什麼是變數，以及如何在程式中使用它們
  • 熟悉變數命名的規則

  • 學會進行型態轉換，靈活處理不同資料型態

透過學習這些內容，你將為 Python 程式設計打下基礎，進一步探索更複雜的應用。

什麼是變數？

變數在 Python 中是參照（reference），它本身並不是儲存資料的實際值，而是指向記憶體中某個物件的位置。當我們將值賦予變數時，變數會參照該值所對應的記憶體位置。。它們允許我們在程式中命名數值或文字，方便重複使用。

x = 10
name = "Alice"
print(x)
print(name)

這裡：

  • x 是一個整數型態的變數，指向 10。
  • name 是一個字串型態的變數，指向 "Alice"。

變數命名規則

在 Python 中，變數命名需遵守以下規則：

  1. 只能包含字母、數字和底線（_）。
  2. 不能以數字開頭。
  3. 不能使用 Python 的保留字（例如 if、for）。
  4. 區分大小寫，例如 myVariable 和 myvariable 是不同的變數。

基本資料型態

Python 支援多種基本資料型態，包括：

  • 整數（int）：如 10、-3
  • 浮點數（float）：如 3.14、-0.5
  • 字串（str）：如 "hello"、'world'
  • 布林值（bool）：如 True、False

型態轉換

在 Python 中，可以使用內建函式進行型態轉換：

  • int()：轉換為整數
  • float()：轉換為浮點數
  • str()：轉換為字串
  • bool()：轉換為布林值
x = "10"
y = int(x)  # 將字串 "10" 轉換為整數 10
z = float(y)  # 將整數 10 轉換為浮點數 10.0
print(y, z)

使用 type 查看型態

我們可以使用 type() 函式檢查變數的資料型態：

x = 42
print(type(x))  # 輸出：<class 'int'>

更多範例：

print(type(3.14))    # <class 'float'>
print(type("hello")) # <class 'str'>
print(type(True))    # <class 'bool'>

使用 id 查看參照位址

每個變數在記憶體中都有唯一的參照位址，可以使用 id() 函式來檢查：

x = 42
y = x
print(id(x))  # 輸出 x 的參照位址
print(id(y))  # 與 x 相同，因為 y 指向同一個物件

改變變數的值會導致其參照位址改變：

x = 42
print(id(x))
x = 43
print(id(x))  # 與之前不同

常見問題與解答

問題 1：為什麼變數名稱不能以數字開頭？

因為以數字開頭的名稱會和數字型態衝突，例如 1var 會被解釋為語法錯誤。

問題 2：如何避免保留字命名問題？

避免使用 Python 的保留字作為變數名稱，可以參考官方文件或使用 IDE 的自動提示功能。

問題 3：什麼是不可變物件？

例如，整數和字串是不可變物件，一旦定義就無法修改。改變變數值時，會建立一個新物件，將變數指向該物件。

挑戰題

挑戰 1：定義並輸出變數 🛈

定義一個整數、一個浮點數、一個字串和一個布林值的變數，然後輸出它們的值和型態。

# 解答
x = 42  # 整數
y = 3.14  # 浮點數
z = "Hello"  # 字串
b = True  # 布林值

print(x, type(x))  # 42 
print(y, type(y))  # 3.14 
print(z, type(z))  # Hello 
print(b, type(b))  # True

挑戰 2：型態轉換 🛈

定義一個變數，將它從整數轉換為字串，然後再轉換為浮點數，最後輸出每一步的結果。

# 解答
x = 10  # 整數
x_str = str(x)  # 轉換為字串
x_float = float(x_str)  # 再轉換為浮點數

print(x, type(x))  # 10 
print(x_str, type(x_str))  # '10' 
print(x_float, type(x_float))  # 10.0

挑戰 3：觀察參照位址 🛈

定義兩個相等的整數變數，檢查它們的參照位址是否相同。嘗試改變其中一個變數的值，然後再次檢查。

# 解答
a = 42
b = 42

print(id(a))  # 輸出 a 的參照位址
print(id(b))  # 與 a 相同

b = 43
print(id(b))  # b 的位址改變

挑戰 5：布林運算 🛈

定義兩個布林值變數，進行與運算（and）、或運算（or）和非運算（not），輸出結果。

# 解答
a = True
b = False

print(a and b)  # 輸出：False
print(a or b)  # 輸出：True
print(not a)  # 輸出：False

挑戰 6：多重型態轉換 🛈

定義一個變數 x = 10.5，將它轉換為整數，再轉換為字串，最後轉換回浮點數，並輸出每一步的型態與值。

# 解答
x = 10.5  # 浮點數
x_int = int(x)  # 轉換為整數
x_str = str(x_int)  # 轉換為字串
x_float = float(x_str)  # 轉回浮點數

print(x, type(x))       # 10.5 
print(x_int, type(x_int))   # 10 
print(x_str, type(x_str))   # '10' 
print(x_float, type(x_float)) # 10.0

挑戰 7：動態變數型態 🛈

定義一個變數，賦值為整數、浮點數和字串，依次輸出變數的型態與值，觀察 Python 如何處理。

# 解答
x = 42  # 整數
print(x, type(x))  # 42 

x = 3.14  # 浮點數
print(x, type(x))  # 3.14 

x = "Hello, Python!"  # 字串
print(x, type(x))  # 'Hello, Python!'

挑戰 8：檢查記憶體參照 🛈

定義兩個變數分別指向相同的浮點數值，檢查它們的記憶體參照位址是否相同。嘗試改變其中一個變數，並重新檢查。

# 解答
a = 3.14
b = 3.14

print(id(a))  # 輸出 a 的參照位址
print(id(b))  # 與 a 相同，因為它們指向相同的物件

b = 3.15  # 改變 b 的值
print(id(a))  # a 的位址保持不變
print(id(b))  # b 的位址改變，指向新物件
