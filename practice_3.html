挑戰題 3（★☆☆）：名字解碼器 ✂️ 🛈

涵蓋主題：字串方法與函式，打包與解包

你拿到一份輸入名單，都是大小寫混亂的名字，請你將其正確格式化為「名 姓」。

"lIOnel, mEssi"
"taYLoR, swIFt"

請你：

  • 使用 split() 拆開姓與名
  • 使用多重指定分別指派給 last 和 first
  • 使用 capitalize() 標準化每個字元
  • 重新輸出為「名 姓」格式
Lionel Messi
Taylor Swift
def fix_name(record):
    last, first = record.split(", ")
    return f"{first.capitalize()} {last.capitalize()}"

names = [
    "lIOnel, mEssi",
    "taYLoR, swIFt"
]

for name in names:
    print(fix_name(name))

涵蓋主題：字典建構式，成員運算符號

for item in orders:
    for k, v in types.items():
        if item in v:
            result[k].append(item)

print(result)

將顧客資料存成類別，每筆資料包含姓名與會員等級，並印出格式化結果。


name,level
Alice,Gold
Bob,Silver


請你：


  • 定義 Customer 類別並初始化姓名與等級
    • 

  • 讀取資料後建立物件
    • 

  • 格式化輸出顧客資訊


Alice (Gold 會員)
Bob (Silver 會員)



  
class Customer:
    def __init__(self, name, level):
        self.name = name
        self.level = level

data = ["Alice,Gold", "Bob,Silver"]
for line in data:
    name, level = line.split(",")
    c = Customer(name, level)
    print(f"{c.name} ({c.level} 會員)")

    if name in online_users:
        print(f"{name} is online")
    else:
        print(f"{name} is offline")

online = ['Amy', 'Bob', 'Dora']
