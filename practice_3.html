挑戰題 7：怪名字清洗器 🧼 🛈

你接手一份包含奇怪名字的名單，例如：

@aMy!
#B_ob23
chA_rLie##

請撰寫一個函式 clean_name()，具備以下功能：

  • 去除所有非英文字母的字元（只保留 a-z 和 A-Z）
  • 將字首轉為大寫，其餘字母小寫（使用 .capitalize()）
  • 若名字長度少於 2 或過長（超過 10 個字母），則回傳 "Invalid"

對以下清單進行處理並列印結果：

["@aMy!", "#B_ob23", "chA_rLie##", "x", "thisisaverylongname"]

Amy
Bob
Charlie
Invalid
Invalid

def clean_name(raw):
    clean = "".join([c for c in raw if c.isalpha()])
    if len(clean) < 2 or len(clean) > 10:
        return "Invalid"
    return clean.capitalize()

names = ["@aMy!", "#B_ob23", "chA_rLie##", "x", "thisisaverylongname"]

for name in names:
    print(clean_name(name))

挑戰題 8：帳號健檢系統 🛡️ 🛈

設計一個帳號健檢工具，幫助使用者確認帳號是否符合格式標準。

usernames = ['goodUser1', 'no space', 'short', 'toolongusername123', 'Valid_User']

請你：

  • 撰寫函式 is_valid_username()，條件如下：
  • 1. 長度需介於 6～15 字元之間
  • 2. 不可包含空格或特殊符號（僅可使用英文字母與數字）
  • 3. 必須至少含有一個英文字母
  • 使用迴圈印出每個帳號是否有效
goodUser1 ✅
no space ❌
short ❌
toolongusername123 ❌
Valid_User ❌

def is_valid_username(username):
    if not (6 <= len(username) <= 15):
        return False
    if not username.isalnum():
        return False
    if not any(c.isalpha() for c in username):
        return False
    return True

usernames = ['goodUser1', 'no space', 'short', 'toolongusername123', 'Valid_User']

for name in usernames:
    status = "✅" if is_valid_username(name) else "❌"
    print(f"{name} {status}")

你已經定義好產品的類別對應如下：

    'drink': ['tea', 'coffee'],
    'food': ['sandwich', 'cake']
}

請你完成程式：

  • 定義輸入清單 orders = ['tea', 'sandwich', 'phone']
  • 建立分類結果字典 result，初始為空清單
  • 只處理在已知類別清單中的項目（忽略未知項目，如 'phone'）
  • 最後輸出分類結果，如：{'drink': ['tea'], 'food': ['sandwich']}
orders = ['tea', 'sandwich', 'phone']
types = {
    'drink': ['tea', 'coffee'],
    'food': ['sandwich', 'cake']
}

result = {k: [] for k in types}

for item in orders:

挑戰題 16：密碼強度檢查器 🔐 🛈

你正在設計一個模組 password_tool.py，它能幫助網站或 App 檢查使用者密碼是否符合強度要求。

請你：

  1. 定義函式 is_strong(password)，檢查密碼是否同時包含：
    • 至少 8 個字元
    • 大寫英文
    • 小寫英文
    • 數字
    • 符號（如 !, @, #, $, %, etc.）
  2. 使用者可匯入此模組並呼叫 is_strong("Hello123!")
  3. 若直接執行此檔案，提示使用者輸入密碼並立即檢查結果

範例：

# 匯入後使用：
>>> import password_tool
>>> password_tool.is_strong("P@ssw0rd")
True

# 直接執行：
請輸入密碼：12345
密碼太弱，請加強！

import string

def is_strong(password):
    return (
        len(password) >= 8 and
        any(c.isupper() for c in password) and
        any(c.islower() for c in password) and
        any(c.isdigit() for c in password) and
        any(c in string.punctuation for c in password)
    )

if __name__ == "__main__":
    pwd = input("請輸入密碼：")
    if is_strong(pwd):
        print("密碼強度足夠 ✅")
    else:
        print("密碼太弱，請加強！ ❌")
