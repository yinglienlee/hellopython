挑戰題 1：神祕訊息解碼器 🕵️‍♂️ 🛈

你收到了一份神祕檔案 secrets.txt，內容如下，每一行都是一段被雜訊干擾的訊息：

Txxhxxixx xxisxx xxaxxsxxexxcrxxexxtxx
Pxxyxxtxxhxxoxxnxx xxixxsxx xxfxuxnxx!
Cxxoxxdxexx xxixxsxx pxxoxxwxexxrfxuxlxx.

但你注意到：每三個字元中，第 1 個字元才是真正的內容，其餘是雜訊！

請你寫一段 Python 程式，讀取檔案後，解碼每一行的訊息，並印出以下內容：

This is a secret
Python is fun!
Code is powerful.

提示：使用 字串切片 與 檔案操作 來完成。

with open("secrets.txt", "r", encoding="utf-8") as f:
    for line in f:
        clean = line.strip()[::3]
        print(clean)
