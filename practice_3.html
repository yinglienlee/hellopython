挑戰題 1：神祕訊息解碼器 🕵️‍♂️ 🛈

你收到了一份神祕檔案 secrets.txt，內容如下，每一行都是一段被雜訊干擾的訊息：

Txxhxxixx xxisxx xxaxxsxxexxcrxxexxtxx
Pxxyxxtxxhxxoxxnxx xxixxsxx xxfxuxnxx!
Cxxoxxdxexx xxixxsxx pxxoxxwxexxrfxuxlxx.

但你注意到：每三個字元中，第 1 個字元才是真正的內容，其餘是雜訊！

請你寫一段 Python 程式，讀取檔案後，解碼每一行的訊息，並印出以下內容：

This is a secret
Python is fun!
Code is powerful.

提示：使用 字串切片 與 檔案操作 來完成。

with open("secrets.txt", "r", encoding="utf-8") as f:
    for line in f:
        clean = line.strip()[::3]
        print(clean)

挑戰題 2：寵物辨識系統 🐶🐱 🛈

你正在設計一套「寵物辨識系統」，幫助飼主確認兩隻動物是不是同一隻、以及牠們是不是同類。

請你完成一個 Pet 類別，每個寵物有名字（name）和種類（species，例如 "dog" 或 "cat"）。接著建立幾個寵物物件，並使用：

  • is 運算子來判斷是否是「同一隻寵物」
  • in 運算子來判斷某種類是否在允許照顧的清單中

模擬下列情境：

  1. 判斷 pet1 和 pet2 是否為同一物件
  2. 確認 pet1 的種類是否在照顧清單 ["dog", "cat", "rabbit"] 中

提示：使用 is 判斷物件身分，in 判斷成員關係。

class Pet:
    def __init__(self, name, species):
        self.name = name
        self.species = species

pet1 = Pet("Buddy", "dog")
pet2 = Pet("Buddy", "dog")
pet3 = pet1  # 指向同一物件

# 身分比較
print(pet1 is pet2)  # False，雖然資料一樣，但是不同物件
print(pet1 is pet3)  # True，同一個物件參考

# 成員測試
allowed_species = ["dog", "cat", "rabbit"]
if pet1.species in allowed_species:
    print(f"{pet1.name} 的種類 {pet1.species} 可接受照顧！")
else:
    print(f"{pet1.name} 的種類 {pet1.species} 不在可照顧清單中。")

挑戰題 3：姓名格式大翻修 ✂️ 🛈

你拿到一份輸入名單，裡面都是使用者亂填的名字資料，例如：

"lIOnel, mEssi"
"taYLoR, swIFt"
"kuRo, miKO"

每一筆都是「姓, 名」格式，大小寫混亂。請你設計一段程式碼，

  • 將姓與名使用 split() 拆開，並使用 **多重指定** 方式指派給 last 和 first
  • 把每個字母都轉為「首字大寫」，也就是 capitalize()
  • 重新組合為「名 姓」格式並印出

加分挑戰：使用函式封裝轉換邏輯，並處理多筆名單！

def fix_name(record):
    last, first = record.split(", ")
    return f"{first.capitalize()} {last.capitalize()}"

names = [
    "lIOnel, mEssi",
    "taYLoR, swIFt",
    "kuRo, miKO"
]

for name in names:
    print(fix_name(name))

挑戰題 4：氣象小幫手 ☀️🌧️ 🛈

你正在開發一個氣象小程式，用來協助使用者查看最近一週的高溫資料。

已知高溫資料如下，請完成下列任務：

[30, 32, 31, 33, 34, 29, 28]
  • 使用 list() 建構這份資料
  • 使用切片技巧取得：
    • 前三天的氣溫
    • 最後三天的氣溫
    • 間隔一天的氣溫（從第 1 天開始，每隔一天擷取一次）

提示：這題重點在於資料建構與切片技巧的靈活運用。

temperatures = list([30, 32, 31, 33, 34, 29, 28])

# 前三天
print("前三天氣溫：", temperatures[:3])

# 最後三天
print("最後三天氣溫：", temperatures[-3:])

# 每隔一天的氣溫
print("隔日氣溫：", temperatures[::2])
