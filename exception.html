例外處理

學習導覽

在這一章，我們將學習如何處理程式執行中的例外情況 (exceptions)。例外處理是撰寫穩健程式碼的基礎，可以防止程式在遇到錯誤時當住。

  • 了解什麼是例外。
  • 學習如何使用 try, except, else 和 finally。
  • 掌握如何自訂例外。

例外處理的基本語法

1. 使用 try 與 except

使用 try 包裹可能發生例外的程式碼，並用 except 處理例外：

try:
    num = int(input("請輸入一個整數: "))
    print(f"你輸入的數字是 {num}")
except ValueError:
    print("輸入的不是有效的整數！")

2. 處理多種例外

如果一個程式可能會引發不同類型的例外，可以使用多個 except 區塊來處理不同的錯誤：

try:
    numerator = int(input("請輸入分子: "))
    denominator = int(input("請輸入分母: "))
    result = numerator / denominator
    print(f"結果是 {result}")
except ValueError:
    print("輸入的不是有效的整數！")
except ZeroDivisionError:
    print("除數不能為零！")

3. 一次捕獲多種類型的例外

可以在同一個 except 區塊內處理多種例外，使用小括號包裹：

try:
    num_list = [1, 2, 3]
    index = int(input("請輸入索引值: "))
    print(f"選取的數字是 {num_list[index]}")
except (ValueError, IndexError):
    print("輸入的索引值無效或超出範圍！")

4. 捕獲所有例外

使用 Exception 可以捕獲所有可能的例外

try:
    file = open("不存在的檔案.txt", "r")
    content = file.read()
except Exception as e:
    print(f"發生錯誤: {e}")

不指定例外也可捕獲所有可能的例外，但應該謹慎使用，以免忽略特定錯誤的詳細資訊。

try:
    file = open("不存在的檔案.txt", "r")
    content = file.read()
except:
    print("發生錯誤")

5. 使用 else 與 finally

else 用於當 try 區塊中沒有發生例外時執行的程式碼，而 finally 則一定會執行：

file = None
try:
    file = open("example.txt", "r")
    content = file.read()
    print(content)
except FileNotFoundError:
    print("檔案不存在！")
else:
    print("檔案讀取成功！")
finally:
    if file:
        file.close()
        print("檔案已關閉。")

if 'file' in locals() 主要用於確保變數 file 存在於當前的區域變數（local variables）中。locals() 回傳當前作用域的所有區域變數，以字典形式表示。

6. 自訂例外

可以建立自訂的例外類別以適應特定需求：

class CustomError(Exception):
    pass

try:
    raise CustomError("這是一個自訂例外！")
except CustomError as e:
    print(f"捕獲自訂例外: {e}")

常見問題與解答

問題 1：為什麼需要例外處理？

例外處理可防止程式因未預期錯誤而終止，並提供機會進行適當的補救行為。

問題 2：可以一次捕獲多種例外嗎？

可以，使用元組列出要捕獲的例外：

try:
    result = 10 / 0
except (ZeroDivisionError, ValueError):
    print("發生錯誤！")

問題 3：如何捕獲所有例外？

使用 except Exception 可以捕獲所有例外，但應謹慎使用以避免隱藏錯誤細節。

try:
    num1 = int(input("請輸入一個整數: "))
    num2 = int(input("請輸入另一個整數: "))
    print(f"你輸入的數字是 {num1}, {num2}")
    print(f"{num1} + {num2} = {num1+num2}")
    print(f"{num1} - {num2} = {num1-num2}")
    print(f"{num1} * {num2} = {num1*num2}")
    print(f"{num1} / {num2} = {num1/num2}")
except ValueError:
    print("輸入錯誤")
except ZeroDivisionError:
    print("除數不可為零")    
except Exception:
    print("程式發生錯誤")

問題 4：如何找出可能遇到的例外？以 int() 為例。

我們可以使用 try-except 來找出 int() 可能拋出的例外類型：

try:
    int("hello")  # 無效的字串輸入
except Exception as e:
    print(f"發生的例外類型: {type(e).__name__}")  # 預期輸出: ValueError

try:
    int([])  # 傳入列表
except Exception as e:
    print(f"發生的例外類型: {type(e).__name__}")  # 預期輸出: TypeError

挑戰題目

挑戰 1：例外處理的基礎 🛈

設計一個程式，要求使用者輸入兩個數字，然後進行除法運算。如果輸入的不是數字或除以零，則捕獲例外並提示錯誤訊息。

# 解答
try:
    a = float(input("請輸入第一個數字: "))
    b = float(input("請輸入第二個數字: "))
    result = a / b
    print(f"結果是: {result}")
except ValueError:
    print("輸入的不是有效數字！")
except ZeroDivisionError:
    print("除數不能為零！")

挑戰 2：多層次例外處理 🛈

設計一個程式，模擬 ATM 提款流程：

  1. 要求使用者輸入提款金額。
  2. 捕獲以下例外情況：
    • 使用者輸入的不是數字。
    • 提款金額小於 0（顯示「金額無效！」）。
    • 提款金額大於帳戶餘額（顯示「餘額不足！」）。
  3. 如果輸入金額正確且可提取，顯示「提款成功！您的餘額是：<剩餘金額>」。
# 解答
balance = 1000  # 帳戶餘額

try:
    amount = float(input("請輸入提款金額: "))
    if amount < 0:
        raise ValueError("金額無效！")
    elif amount > balance:
        raise ValueError("餘額不足！")
    else:
        balance -= amount
        print(f"提款成功！您的餘額是：{balance}")
except ValueError as e:
    print(e)
except Exception:
    print("發生未知錯誤！")

挑戰 3：自訂例外 🛈

建立一個自訂例外類別，模擬當使用者年齡小於 18 歲時，拋出 "未滿 18 歲不可註冊" 的例外。

# 解答
class AgeRestrictionError(Exception):
    pass

try:
    age = int(input("請輸入您的年齡: "))
    if age < 18:
        raise AgeRestrictionError("未滿 18 歲不可註冊！")
    print("註冊成功！")
except AgeRestrictionError as e:
    print(e)
except ValueError:
    print("請輸入有效的數字！")

挑戰 4：分數計算 🛈

設計一個程式來計算兩個分數的加總，並處理可能的例外：

  1. 要求使用者輸入分數的分子與分母。
  2. 捕獲以下例外情況：
    • 輸入的不是數字（顯示「請輸入有效的數字！」）。
    • 分母為零（顯示「分母不能為零！」）。
  3. 若輸入有效，計算並顯示結果，例如：分數加總結果為 7/4。
# 解答
try:
    num1 = int(input("請輸入第一個分數的分子: "))
    den1 = int(input("請輸入第一個分數的分母: "))
    num2 = int(input("請輸入第二個分數的分子: "))
    den2 = int(input("請輸入第二個分數的分母: "))
    if den1 == 0 or den2 == 0:
        raise ZeroDivisionError("分母不能為零！")
    numerator = num1 * den2 + num2 * den1
    denominator = den1 * den2
    print(f"分數加總結果為 {numerator}/{denominator}")
except ValueError:
    print("請輸入有效的數字！")
except ZeroDivisionError as e:
    print(e)

挑戰 5：例外處理中的多個 except 🛈

撰寫一個程式，模擬簡單的計算機操作：

  1. 要求使用者輸入兩個數字以及一個運算符號（+、-、*、/）。
  2. 捕獲以下例外：
    • 使用者輸入的不是數字（顯示「輸入無效！」）。
    • 使用者嘗試除以零（顯示「不能除以零！」）。
    • 無效的運算符號（顯示「不支援的運算符號！」）。
  3. 如果輸入正確，計算並輸出結果。
# 解答
try:
    num1 = float(input("請輸入第一個數字: "))
    num2 = float(input("請輸入第二個數字: "))
    operator = input("請輸入運算符號（+、-、*、/）: ")
    if operator == "+":
        print(f"結果是: {num1 + num2}")
    elif operator == "-":
        print(f"結果是: {num1 - num2}")
    elif operator == "*":
        print(f"結果是: {num1 * num2}")
    elif operator == "/":
        if num2 == 0:
            raise ZeroDivisionError("不能除以零！")
        print(f"結果是: {num1 / num2}")
    else:
        raise ValueError("不支援的運算符號！")
except ValueError as e:
    print(e)
except ZeroDivisionError as e:
    print(e)

挑戰 6：自訂例外處理邏輯 🛈

設計一個程式來模擬簡單的密碼驗證系統：

  1. 撰寫一個自訂例外類別 PasswordTooShortError。
  2. 要求使用者輸入密碼，若密碼長度小於 8 個字元，拋出該例外並提示「密碼太短！」。
  3. 若密碼有效，顯示「密碼設定成功！」。
# 解答
class PasswordTooShortError(Exception):
    pass

try:
    password = input("請輸入密碼: ")
    if len(password) < 8:
        raise PasswordTooShortError("密碼太短！")
    print("密碼設定成功！")
except PasswordTooShortError as e:
    print(e)

挑戰 7：巢狀例外處理 🛈

設計一個程式，模擬巢狀例外的處理過程：

  1. 撰寫一個函式模擬數字相加的過程。
  2. 捕獲以下例外：
    • 輸入的不是有效數字（顯示「輸入錯誤！」）。
  3. 若無例外，顯示運算結果。
# 解答
def add_numbers(a, b):
    try:
        return a + b
    except TypeError:
        print("輸入錯誤！")

try:
    num1 = input("請輸入第一個數字: ")
    num2 = input("請輸入第二個數字: ")
    result = add_numbers(int(num1), int(num2))
    print(f"結果是: {result}")
except ValueError:
    print("請輸入有效的數字！")

挑戰 8：條件式錯誤驗證 🛈

設計一個程式來驗證使用者輸入的年齡是否符合條件：

  1. 要求使用者輸入年齡。
  2. 捕獲以下例外情況：
    • 輸入的不是數字（顯示「請輸入有效的數字！」）。
    • 年齡小於 0 或超過 120（顯示「年齡範圍無效！」）。
  3. 若輸入有效，顯示「年齡有效！」。
# 解答
try:
    age = int(input("請輸入您的年齡: "))
    if age < 0 or age > 120:
        raise ValueError("年齡範圍無效！")
    print("年齡有效！")
except ValueError as e:
    print(e)
