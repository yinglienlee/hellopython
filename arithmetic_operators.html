算術運算符號與運算優先順序

學習導覽

算術運算符號

Python 提供了基本的算術運算符號，用於處理數字運算：

  • +：加法，例如 3 + 2 結果為 5
  • -：減法，例如 3 - 2 結果為 1
  • *：乘法，例如 3 * 2 結果為 6
  • /：除法，例如 3 / 2 結果為 1.5
  • //：整數除法（取商），例如 3 // 2 結果為 1
  • %：取餘數，例如 3 % 2 結果為 1
  • **：次方，例如 3 ** 2 結果為 9

指定運算符號

Python 提供了用於簡化變數操作的指定運算符號：

  • +=：加等於，例如 x += 2 等價於 x = x + 2
  • -=：減等於，例如 x -= 2 等價於 x = x - 2
  • *=：乘等於，例如 x *= 2 等價於 x = x * 2
  • /=：除等於，例如 x /= 2 等價於 x = x / 2
  • //=：整數除等於，例如 x //= 2 等價於 x = x // 2
  • %=：取餘數等於，例如 x %= 2 等價於 x = x % 2
  • **=：次方等於，例如 x **= 2 等價於 x = x ** 2

運算優先順序

Python 的運算符號具有優先順序，以下為基本規則（從高到低）：

  1. 括號 ()
  2. 指數 **
  3. 正負號 +x、-x
  4. 乘除取餘數 *、/、//、%
  5. 加減 +、-

範例：

result = 3 + 2 * 2  # 結果為 7
result = (3 + 2) * 2  # 結果為 10

用在字串的運算符號

某些運算符號可以用於字串操作：

  • +：連接字串，例如 "Hello" + " World" 結果為 "Hello World"
  • *：重複字串，例如 "Ha" * 3 結果為 "HaHaHa"

常見問題與解答

問題 1：為什麼 / 與 // 的結果不同？

/ 會回傳浮點數，而 // 會回傳整數，捨去小數部分。

問題 2：字串運算會導致錯誤嗎？

當兩個運算元的型態不一致時，可能會導致錯誤。例如，數字不能與字串直接相加。

問題 3：使用 ** 運算符號有什麼限制？

次方運算符號 ** 可以用於整數與浮點數，但當基數或指數非常大時，可能會導致運算時間變長或產生溢位錯誤。例如：

2 ** 1000  # 非常大的數字，可能會消耗大量記憶體

問題 4：如何避免運算優先順序帶來的誤解？

假設有兩門課程的分數和權重，例如：

數學分數 = 80
英文分數 = 90
數學權重 = 3
英文權重 = 2

平均分數 = 數學分數 * 數學權重 + 英文分數 * 英文權重 / 數學權重 + 英文權重
print("錯誤的平均分數:", 平均分數)

Python 會先計算 英文分數 * 英文權重 / 數學權重，導致錯誤。這樣的計算方式沒有按照加權平均的正確公式來計算。

數學分數 = 80
英文分數 = 90
數學權重 = 3
英文權重 = 2

平均分數 = (數學分數 * 數學權重 + 英文分數 * 英文權重) / (數學權重 + 英文權重)
print("正確的平均分數:", 平均分數)

加上括號 (數學分數 * 數學權重 + 英文分數 * 英文權重)，確保加總後再除以總權重。避免 Python 依照預設的運算順序 (* 和 / 先計算) 而導致錯誤的結果。

問題 5：在 Python 中，可以用運算符號來處理布林值嗎？

可以。布林值 True 與 False 會分別被當作 1 和 0。例如：

result = True + 2  # 結果為 3
result = False * 5  # 結果為 0

問題 6：如何安全地進行除法，避免被 0 除錯誤？

在執行除法運算之前，應檢查除數是否為 0。例如：

numerator = 10
denominator = 0
if denominator != 0:
    result = numerator / denominator
else:
    print("除數不能為 0")

挑戰 1：計算數學表達式 🛈

讓使用者輸入三個數字，計算其加減乘除的結果，並輸出。

# 解答
print("請輸入三個數字：")
num1 = int(input("第一個數字："))
num2 = int(input("第二個數字："))

print("減法結果：", num1 - num2 - num3)
print("乘法結果：", num1 * num2 * num3)

    print("除法結果：", num1 / num2 / num3)
else:
    print("無法進行除法，因為除數為 0")

挑戰 2：字串操作 🛈

# 解答

times = int(input("請輸入一個數字："))

挑戰 3：綜合運算 🛈

定義兩個整數，分別進行加減乘除、取餘數與平方，並輸出每個運算的結果。

# 解答
num1 = int(input("第一個整數："))
num2 = int(input("第二個整數："))
print("加法結果：", num1 + num2)
print("減法結果：", num1 - num2)
print("乘法結果：", num1 * num2)
if num2 != 0:
    print("除法結果：", num1 / num2)
    print("取餘數結果：", num1 % num2)
else:
    print("無法進行除法或取餘數，因為除數為 0")
print("平方結果：", num1 ** num2)

挑戰 4：運算優先順序練習 🛈

請建立一個程式，讓使用者輸入三個數字，分別進行以下運算：

  1. 不加括號，依照預設運算優先順序進行計算（如加法與乘法的混合運算）。
  2. 加入括號，改變運算的優先順序。

輸出兩種計算結果，並觀察結果的差異，理解括號對運算優先順序的影響。

# 解答
num1 = int(input("第一個數字："))
num2 = int(input("第二個數字："))
num3 = int(input("第三個數字："))
print("不加括號的結果：", num1 + num2 * num3)

挑戰 5：指定運算符號應用 🛈

定義一個初始值，使用指定運算符號（+=、*= 等）進行多次運算，並輸出每一步的結果。

# 解答
value = int(input("請輸入初始值："))
value += 5
print("加 5 的結果：", value)
value *= 2
print("乘以 2 的結果：", value)
value -= 3
print("減 3 的結果：", value)
value //= 4
print("整數除以 4 的結果：", value)

挑戰 6：檢查奇數或偶數 🛈

讓使用者輸入一個整數，使用取餘數運算符號（%）檢查該數字是奇數還是偶數，並輸出結果。

# 解答
num = int(input("請輸入一個整數："))
if num % 2 == 0:
    print("這是一個偶數")
else:
    print("這是一個奇數")

挑戰 7：安全除法 🛈

讓使用者輸入兩個整數，計算第一個數字除以第二個數字的結果。如果除數為 0，顯示錯誤訊息並提示重新輸入。

# 解答
numerator = int(input("請輸入被除數："))
while True:
    denominator = int(input("請輸入除數（不能為 0）："))
    if denominator != 0:
        result = numerator / denominator
        print("計算結果是：", result)
        break
    else:
        print("除數不能為 0，請重新輸入！")
