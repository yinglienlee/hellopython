身分與成員運算符號

學習導覽

在本章中，我們將深入探討 Python 中的身分與成員運算符號，這些工具可以幫助我們檢查物件的唯一性及其內容。

  • 身分運算符號：is 和 is not。
  • 成員運算符號：in 和 not in。

理解這些運算符號的用法，將幫助你更靈活地進行條件判斷與物件操作。

身分運算符號

is 和 is not

身分運算符號用於比較兩個物件的記憶體位址是否相同。

a = [1, 2, 3]
b = a
c = [1, 2, 3]

print(a is b)  # 輸出 True，因為 a 和 b 是同一物件
print(a is c)  # 輸出 False，因為 a 和 c 雖然內容相同，但不是同一物件
print(a is not c)  # 輸出 True

請注意，is 與 == 的功能不同：is 檢查物件身分，而 == 檢查物件值是否相等。

成員運算符號

in 和 not in

成員運算符號用於檢查一個值是否存在於容器中，例如串列、字典或字串。

numbers = [1, 2, 3, 4]

print(3 in numbers)  # 輸出 True，因為 3 在 numbers 中
print(5 not in numbers)  # 輸出 True，因為 5 不在 numbers 中

這些運算符號同樣適用於字串：

text = "hello world"

print("hello" in text)  # 輸出 True
print("Python" not in text)  # 輸出 True

常見問題與解答

問題 1：什麼時候應該使用 is 而不是 ==？

is 用於比較物件身分，適用於檢查是否為同一物件；== 則用於比較物件值。

問題 2：in 是否能檢查字典的值？

in 檢查的是字典的鍵，若需檢查值，可以使用 value in dictionary.values()。

data = {"name": "Alice", "age": 25}

print("name" in data)  # 輸出 True
print("Alice" in data.values())  # 輸出 True

挑戰 1：檢查物件身分 🛈

撰寫程式，建立兩個串列並檢查它們是否為同一物件。

# 範例輸入
list1 = [1, 2, 3]
list2 = [1, 2, 3]
# 預期輸出
list1 is list2：False
# 解答
list1 = [1, 2, 3]
list2 = [1, 2, 3]
print(f"list1 is list2：{list1 is list2}")

挑戰 2：檢查成員 🛈

撰寫程式，檢查一個字串是否包含某些特定單詞。

# 範例輸入
text = "The quick brown fox"
word = "quick"
# 預期輸出
"quick" 是否存在：True
# 解答
text = "The quick brown fox"
word = "quick"
print(f'"{word}" 是否存在：{word in text}')

挑戰 3：條件篩選 🛈

設計程式，從一個串列中篩選出所有偶數，並檢查特定值是否在結果中。

# 範例輸入
numbers = [1, 2, 3, 4, 5, 6]
value = 4
# 預期輸出
篩選後的偶數：[2, 4, 6]
4 是否存在：True
# 解答
numbers = [1, 2, 3, 4, 5, 6]
value = 4
evens = [num for num in numbers if num % 2 == 0]
print(f"篩選後的偶數：{evens}")
print(f"{value} 是否存在：{value in evens}")

挑戰 4：檢查字典中的鍵與值 🛈

設計一個程式，檢查一個字典中是否存在特定的鍵與值。

# 範例輸入
data = {"name": "Alice", "age": 30}

"name" 是否為鍵：True
"Alice" 是否為值：True
# 解答
data = {"name": "Alice", "age": 30}
print(f'"name" 是否為鍵：{"name" in data}')
print(f'"Alice" 是否為值：{"Alice" in data.values()}')

挑戰 5：比較物件身分與內容 🛈

建立兩個內容相同但記憶體位置不同的串列，使用 is 和 == 比較它們，並解釋結果。

# 範例輸入
a = [1, 2, 3]
b = [1, 2, 3]
# 預期輸出
a == b：True
a is b：False
# 解答
a = [1, 2, 3]
b = [1, 2, 3]
print(f"a == b：{a == b}")
print(f"a is b：{a is b}")

挑戰 6：判斷成員的多層結構 🛈

給定一個巢狀串列，檢查某個值是否存在於其中的任何一層。

# 範例輸入
nested_list = [1, [2, [3, 4]], 5]
# 預期輸出
3 是否存在：True
# 解答
def contains_value(nested, target):
    for item in nested:
        if type(item) == list:  # 使用 type 檢查是否是 list
            if contains_value(item, target):
		return True
        elif item == target:
            return True
    return False

nested_list = [1, [2, [3, 4]], 5]
value = 3
print(f"{value} 是否存在：{contains_value(nested_list, value)}")

挑戰 7：清單移除重複並檢查成員 🛈

給定一個包含重複元素的串列，移除重複後檢查某個值是否仍然存在。

# 範例輸入
data = [1, 2, 2, 3, 4, 4, 5]
# 預期輸出
清單移除重複後：{1, 2, 3, 4, 5}
5 是否存在：True
# 解答
data = [1, 2, 2, 3, 4, 4, 5]
unique_data = set(data)
value = 5
print(f"清單移除重複後：{unique_data}")
print(f"{value} 是否存在：{value in unique_data}")
