條件判斷

學習導覽

條件判斷是 Python 程式設計的基礎之一。它讓程式能夠根據不同的情況執行不同的指令，從而使程式更加靈活和智能。本章內容將涵蓋：

  • 使用 if、elif 和 else 關鍵字。
  • 理解條件判斷的邏輯結構。
  • 巢狀條件判斷的使用方式。
  • 學會處理常見的條件判斷問題。

條件判斷語法

1. 使用 if

if 是條件判斷的核心，用來檢查條件是否成立：

x = 10
if x > 5:
    print("x 大於 5")

如果條件 x > 5 為真，將輸出：
x 大於 5

上述程式碼的流程圖如下。

					flowchart TD
						Start --> Set_x["x = 10"]
						Set_x --> Check_x{"Is x > 5?"}
						Check_x -- Yes --> Print["print('x 大於 5')"]						
						Print --> End
						Check_x -- No --> End

2. 使用 elif

elif（縮寫自 "else if"）用於檢查額外的條件：

x = 10
if x > 15:
    print("x 大於 15")
elif x > 5:
    print("x 大於 5 但小於等於 15")

當 x > 15 不成立，但 x > 5 成立時，輸出：
x 大於 5 但小於等於 15

上述程式碼的流程圖如下。

					flowchart TD
						A[Start] --> B[x = 10]
						B --> C{x > 15?}
						C -->|Yes| D["print('x 大於 15')"]
						C -->|No| E{x > 5?}
						E -->|Yes| F["print('x 大於 5 但小於等於 15')"]
						E -->|No| G[End]
						D --> G
						F --> G

3. 使用 else

else 用於處理所有未被上述條件涵蓋的情況：

x = 3
if x > 5:
    print("x 大於 5")
else:
    print("x 小於或等於 5")

當 x > 5 不成立時，輸出：
x 小於或等於 5

上述程式碼的流程圖如下。

					flowchart TD
						A[Start] --> B[x = 3]
						B --> C{x > 5?}
						C -->|Yes| D["print('x 大於 5')"]
						C -->|No| E["print('x 小於或等於 5')"]
						D --> F[End]
						E --> F

4. 巢狀條件判斷

可以在條件內再巢狀另一個條件判斷：

x = 10
if x > 5:
    if x % 2 == 0:
        print("x 是一個偶數且大於 5")

如果條件全部成立，輸出：
x 是一個偶數且大於 5

上述程式碼的流程圖如下。

					flowchart TD
						A[Start] --> B[x = 10]
						B --> C{x > 5?}
						C -->|Yes| D{x % 2 == 0?}
						D -->|Yes| E["print('x 是一個偶數且大於 5')"]
						D -->|No| F[End]
						C -->|No| F
						E --> F

常見問題與解答

問題 1：為什麼條件內的程式碼要縮排？

Python 使用縮排來區分條件判斷內的程式碼。如果忘記縮排，會出現 IndentationError。

# 正確
if True:
    print("這是正確的縮排")

# 錯誤
if True:
print("這會導致錯誤")

問題 2：可以同時檢查多個條件嗎？

可以使用邏輯運算符號 and 和 or：

x = 10
if x > 5 and x < 15:
    print("x 在 5 和 15 之間")

問題 3：如何簡化單行條件判斷？

可以使用單行格式：

print("x 大於 5") if x > 5 else print("x 小於或等於 5")

挑戰題目

挑戰 1：判斷奇偶數 🛈

寫一個程式，輸入一個數字並判斷它是奇數還是偶數。

# 解答
number = int(input("請輸入一個數字："))
if number % 2 == 0:
    print(f"{number} 是偶數")
else:
    print(f"{number} 是奇數")

挑戰 2：多條件檢查 🛈

設計一個程式，檢查一個數字是否：

  • 小於 0
  • 在 0 到 100 之間
  • 大於 100
# 解答
number = int(input("請輸入一個數字："))
if number < 0:
    print("數字小於 0")
elif 0 <= number <= 100:
    print("數字在 0 到 100 之間")
else:
    print("數字大於 100")

挑戰 3：巢狀條件 🛈

撰寫一個程式，輸入三個數字並找出最大值。

# 解答
num1 = int(input("請輸入第一個數字："))
num2 = int(input("請輸入第二個數字："))
num3 = int(input("請輸入第三個數字："))

if num1 >= num2:
    if num1 >= num3:
        print(f"最大值是 {num1}")
    else:
        print(f"最大值是 {num3}")
else:
    if num2 >= num3:
        print(f"最大值是 {num2}")
    else:
        print(f"最大值是 {num3}")

挑戰 4：判斷分數等級 🛈

設計一個程式，讓使用者輸入分數，並根據以下規則判斷等級：

  • 90 分以上為 A 級
  • 80 分到 89 分為 B 級
  • 70 分到 79 分為 C 級
  • 60 分到 69 分為 D 級
  • 60 分以下為 F 級
# 解答
score = int(input("請輸入分數："))
if score >= 90:
    print("A 級")
elif score >= 80:
    print("B 級")
elif score >= 70:
    print("C 級")
elif score >= 60:
    print("D 級")
else:
    print("F 級")

挑戰 5：簡單的計算機 🛈

撰寫一個程式，讓使用者輸入兩個數字和一個運算符號（+、-、*、/），然後根據使用者的輸入進行相應的運算，並輸出結果。

# 解答
num1 = float(input("請輸入第一個數字："))
num2 = float(input("請輸入第二個數字："))
operator = input("請輸入運算符號（+、-、*、/）：")

if operator == "+":
    print(f"結果是：{num1 + num2}")
elif operator == "-":
    print(f"結果是：{num1 - num2}")
elif operator == "*":
    print(f"結果是：{num1 * num2}")
elif operator == "/":
    if num2 != 0:
        print(f"結果是：{num1 / num2}")
    else:
        print("錯誤：除數不能為 0")
else:
    print("無效的運算符號")

挑戰 6：檢查三角形 🛈

設計一個程式，讓使用者輸入三邊長，檢查它們是否能構成一個有效的三角形（任意兩邊之和大於第三邊）。

# 解答
a = float(input("請輸入第一邊長："))
b = float(input("請輸入第二邊長："))
c = float(input("請輸入第三邊長："))

if a + b > c and a + c > b and b + c > a:
    print("可以構成三角形")
else:
    print("不能構成三角形")

挑戰 7：閏年判斷 🛈

撰寫一個程式，讓使用者輸入年份，判斷是否為閏年（能被 4 整除但不能被 100 整除，或者能被 400 整除）。

# 解答
year = int(input("請輸入年份："))

if (year % 4 == 0 and year % 100 != 0) or (year % 400 == 0):
    print(f"{year} 是閏年")
else:
    print(f"{year} 不是閏年")

挑戰 8：巢狀條件 - 找最小值 🛈

設計一個程式，讓使用者輸入三個數字，使用巢狀條件判斷並輸出最小值。

# 解答

if num1 <= num2:

else:
    if num2 <= num3:
        print(f"最小值是 {num2}")
    else:
        print(f"最小值是 {num3}")
