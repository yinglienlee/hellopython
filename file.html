基本檔案操作

1. 開啟檔案

使用 open() 函式開啟檔案：

# 開啟檔案進行讀取
file = open("example.txt", "r")
# 執行檔案操作
file.close()  # 關閉檔案

2. 自動管理檔案的開啟與關閉

建議使用 with 語法來自動管理檔案：

with open("example.txt", "r") as file:
    content = file.read()
    print(content)

如果不關閉檔案，可能會導致以下問題：

  • 資源浪費：檔案在開啟時會佔用系統資源，若不及時關閉，這些資源會一直被占用。當開啟的檔案數量達到系統上限時，將無法再打開新的檔案。
  • 資料遺失或不一致：在寫入檔案時，資料通常先被存放在緩衝區，等到檔案關閉時才真正寫入磁碟。如果不關閉檔案，緩衝區中的資料可能無法正確寫入，導致資料遺失或檔案內容不一致。
  • 檔案鎖定：某些作業系統或應用程式在檔案開啟後會鎖定該檔案，若不關閉，其他程式可能無法讀取或寫入這個檔案。
  • 程式不穩定：長期未關閉檔案可能會導致程式內部的資源管理出現問題，進而引發記憶體流失或其他意料之外的錯誤，影響程式穩定性。

3. 檔案模式

open() 的模式參數決定了檔案的操作方式：

  • "r"：唯讀模式。
  • "w"：寫入模式，會清空檔案內容。
  • "a"：附加模式，將內容寫入檔案末尾。
  • "rb"、"wb"：以二進位模式讀寫。

讀取檔案

1. 使用 read()

讀取檔案的所有內容：

with open("example.txt", "r") as file:
    content = file.read()
    print(content)

2. 使用 readline()

一次讀取一行：

with open("example.txt", "r") as file:
    line = file.readline()
    while line:
        print(line.strip())
        line = file.readline()

3. 使用 readlines()

讀取所有行並回傳串列：

with open("example.txt", "r") as file:
    lines = file.readlines()
    for line in lines:
        print(line.strip())

寫入檔案

1. 使用 write()

將內容寫入檔案：

with open("example.txt", "w") as file:
    file.write("Hello, World!\n")

2. 使用 writelines()

寫入多行內容：

lines = ["Line 1\n", "Line 2\n", "Line 3\n"]
with open("example.txt", "w") as file:
    file.writelines(lines)

例外處理

1. 檔案操作中的例外

處理檔案時可能會遇到例外，例如檔案不存在或無法寫入。

try:
    with open("nonexistent.txt", "r") as file:
        content = file.read()
except FileNotFoundError:
    print("檔案不存在！")

2. 確保檔案正確關閉

雖然 with 語法通常能自動處理，但也可以使用 finally 確保檔案被關閉：

file = None
try:
    file = open("example.txt", "r")
    content = file.read()
finally:
    if 'file' in locals():
        file.close()

不同平台的換行符號

在檔案操作中，換行符號的使用因操作系統而異。這些差異可能會影響到檔案的可讀性或跨平台的相容性，因此了解它們非常重要。

1. 常見的換行符號

  • \n（LF, Line Feed）：用於 Unix/Linux 和 macOS 系統。
  • \r\n（CRLF, Carriage Return + Line Feed）：用於 Windows 系統。
  • \r（CR, Carriage Return）：舊版的 macOS 使用，但現已被 LF 替代。

2. 範例

以下範例展示了不同平台的換行符號對檔案輸出的影響：

# Windows 系統換行符號
with open("example_windows.txt", "w", newline="\r\n") as file:
	file.write("第一行\n第二行")

# Unix/Linux/macOS 系統換行符號
with open("example_unix.txt", "w", newline="\n") as file:
	file.write("第一行\n第二行")

3. 跨平台注意事項

在處理檔案時，建議使用 Python 的內建功能來自動處理換行符號，以確保跨平台的相容性：

with open("example.txt", "w", newline=None) as file:
	file.write("第一行\n第二行")

上述程式會根據操作系統自動應用適當的換行符號。

常見問題與解答

問題 1：可以同時讀取和寫入檔案嗎？

可以，使用模式 "r+" 即可同時讀寫檔案。

問題 2：如何檢查檔案是否存在？

可以使用 os.path.exists()：

import os

if os.path.exists("example.txt"):
    print("檔案存在！")
else:
    print("檔案不存在！")

挑戰 4：搜尋檔案中的關鍵字 🛈

撰寫程式，讓使用者輸入關鍵字，並搜尋檔案中是否包含該關鍵字，輸出該關鍵字所在的行數。

# 範例輸入
# example.txt 檔案內容：
# Hello, this is a test file.
# Python is great.
# File operations are useful.

keyword = "Python"
# 預期輸出
關鍵字 "Python" 出現在第 2 行。
# 解答
file_name = "example.txt"
keyword = input("請輸入關鍵字：")

with open(file_name, "r") as file:
    for line_number, line in enumerate(file, start=1):
        if keyword in line:
            print(f"關鍵字 \"{keyword}\" 出現在第 {line_number} 行。")

挑戰 5：合併多個檔案 🛈

設計一個程式，將多個文字檔案的內容合併為一個新檔案。

# 範例輸入
# file1.txt 和 file2.txt 的內容分別為：
# file1.txt: Hello
# file2.txt: World
# 預期輸出
# 合併後的新檔案內容：
# Hello
# World
# 解答
files_to_merge = ["file1.txt", "file2.txt"]
output_file = "merged.txt"

with open(output_file, "w") as outfile:
    for file_name in files_to_merge:
        with open(file_name, "r") as infile:
            outfile.write(infile.read() + "\n")

print(f"檔案已合併為 {output_file}。")

挑戰 6：檢查檔案大小 🛈

撰寫程式，檢查指定檔案的大小，並輸出檔案大小（以 KB 為單位）。

# 範例輸入

print(f"檔案大小：{file_size:.2f} KB")

挑戰 7：移除檔案中的空行 🛈

設計一個程式，讀取文字檔案，移除所有的空行，並將結果存入新檔案。

# 範例輸入
# example.txt 原內容：
# Line 1
#
# Line 2
#
# 預期輸出（新檔案內容）：

挑戰 8：檔案加密與解密 🛈

撰寫一個簡單的程式，將檔案中的內容進行加密（如每個字母移位 3 個位置），並能將加密內容解密回原文。

# 範例輸入
# 原內容：
# Hello
# 加密後內容：
# Khoor
# 解密後內容：
# Hello
# 解答
# 加密函式
def encrypt(text, shift):
    encrypted = ""
    for char in text:
        if char.isalpha():  # 只處理字母
            base = ord('A') if char.isupper() else ord('a')
            encrypted += chr((ord(char) - base + shift) % 26 + base)
        else:
            encrypted += char
    return encrypted

# 解密函式
def decrypt(text, shift):
    return encrypt(text, -shift)

# 範例檔案加密
with open("example.txt", "r") as file:
    original_content = file.read()

encrypted_content = encrypt(original_content, 3)
with open("encrypted.txt", "w") as file:
    file.write(encrypted_content)

# 範例檔案解密
with open("encrypted.txt", "r") as file:
    encrypted_content = file.read()

decrypted_content = decrypt(encrypted_content, 3)
print("加密後內容：", encrypted_content)
print("解密後內容：", decrypted_content)
