四大資料結構建構式

學習導覽

本章節將介紹 Python 中四種常見的資料結構：list、set、tuple 和 dict，並說明它們的建構式與基本用法。

  • 什麼是 list、set、tuple 和 dict？
  • 如何使用建構式建立這些資料結構。
  • 各種資料結構的適用場景。

資料結構建構式的基本概念

1. list 建構式

使用 list() 或方括號建立可變的有序資料集合：

# 使用 list 建構式
my_list = list((1, 2, 3))
print(my_list)  # 輸出：[1, 2, 3]

# 使用方括號
my_list2 = [4, 5, 6]
print(my_list2)  # 輸出：[4, 5, 6]

2. set 建構式

使用 set() 建立無序且不重複的資料集合：

# 使用 set 建構式
my_set = set([1, 2, 2, 3])
print(my_set)  # 輸出：{1, 2, 3}

3. tuple 建構式

使用 tuple() 或小括號建立不可變的有序資料集合：

# 使用 tuple 建構式
my_tuple = tuple([1, 2, 3])
print(my_tuple)  # 輸出：(1, 2, 3)

# 使用小括號
my_tuple2 = (4, 5, 6)
print(my_tuple2)  # 輸出：(4, 5, 6)

4. dict 建構式

使用 dict() 或大括號建立鍵值組的資料結構：

# 使用 dict 建構式
my_dict = dict([("a", 1), ("b", 2)])
print(my_dict)  # 輸出：{"a": 1, "b": 2}

# 使用大括號
my_dict2 = {"x": 10, "y": 20}
print(my_dict2)  # 輸出：{"x": 10, "y": 20}

常見問題與解答

問題 1：什麼時候使用 list？

當需要儲存可變的有序資料時，例如多個數字或字串。

問題 2：為什麼 set 中沒有重複元素？

set 是無序且不重複的集合，會自動去除重複值。

問題 3：tuple 和 list 有什麼不同？

tuple 是不可變的，適合儲存固定內容，而 list 是可變的。

# 建立 list、set 和 tuple
list_data = [1, 2, 3, 4, 5]
set_data = {1, 2, 3, 4, 5}
tuple_data = (1, 2, 3, 4, 5)

建立一個 dict，包含三個鍵值組，並嘗試新增、更新與刪除鍵值。

# 操作字典
dict_data = {"name": "Alice", "age": 25, "city": "Taipei"}

# 新增
dict_data["job"] = "Engineer"

dict_data["age"] = 26

挑戰 3：結合多種資料結構 🛈

# 儲存多位學生的資料

挑戰 4：轉換資料結構 🛈

給定一個 list，將其轉換為 set 和 tuple，並列印轉換後的結果。

# 原始列表
original_list = [1, 2, 2, 3, 4, 4, 5]

# 轉換為 set 和 tuple
converted_set = set(original_list)
converted_tuple = tuple(original_list)

print("Original List:", original_list)
print("Converted Set:", converted_set)
print("Converted Tuple:", converted_tuple)

挑戰 5：移除重複元素 🛈

設計一個程式，將一個包含重複元素的 list 轉換為不包含重複元素的 set，再轉回 list 並保持原順序。

# 原始列表
original_list = [1, 2, 2, 3, 4, 4, 5]

# 移除重複並保持順序
unique_list = list(dict.fromkeys(original_list))

print("Unique List:", unique_list)

挑戰 6：合併與更新字典 🛈

給定兩個 dict，將它們合併，若有相同的鍵，保留較大的值。

# 合併兩個字典
dict1 = {"a": 1, "b": 2}
dict2 = {"b": 3, "c": 4}

merged = {}

for k in dict1.keys() | dict2.keys():
    merged[k] = max(dict1.get(k, -float('inf')), dict2.get(k, -float('inf')))	
print(merged)

# 使用字典推導式		
merged = {k: max(dict1.get(k, -float('inf')), dict2.get(k, -float('inf'))) for k in dict1.keys() | dict2.keys()}
print(merged)

挑戰 7：排序元組中的數字 🛈

給定一個由多個數字組成的 tuple，將其排序後轉換為 list 並列印。

# 原始元組
unsorted_tuple = (34, 12, 56, 78, 1)

# 排序並轉換為列表
sorted_list = sorted(unsorted_tuple)

print("Sorted List:", sorted_list)

挑戰 8：統計字母出現次數 🛈

設計一個程式，讓使用者輸入一段文字，計算每個字母（a-z）的出現次數，並以字母順序列印結果，忽略大小寫與非字母字元。

# 讓使用者輸入文字
text = input("請輸入一段文字: ").lower()  # 轉換為小寫

# 建立字母計數字典
letter_counts = {}

for char in text:
    if 'a' <= char <= 'z':  # 只計算英文字母
        letter_counts[char] = letter_counts.get(char, 0) + 1

# 依照字母順序排序並輸出
print("字母出現次數：")
for letter in sorted(letter_counts):
    print(f"{letter}: {letter_counts[letter]}")
