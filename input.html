介紹 input 函數

本章節將帶你深入了解 Python 中的 input() 函數，並學會以下內容：

  • 如何透過提示訊息引導使用者輸入

  • 利用 input() 實作多次輸入與多輸入處理

完成本章節後，你將能設計互動式程式，並熟練運用 input() 函數與使用者互動。

使用 input 函數

以下是一個簡單的範例：

name = input("請輸入你的名字：")
print("你好, " + name + "!")

當執行此程式時，程式會等待使用者輸入名字，然後將名字儲存到變數 name，最後顯示歡迎訊息。

input 函數的特性

  • 取得的輸入值型態為字串（str）。即使使用者輸入的是數字，也會以字串形式儲存。
  • 可以透過在括號內提供字串來顯示提示訊息。

範例：處理數字輸入

如果需要數字型態的輸入，需手動進行型態轉換：

age = input("請輸入你的年齡：")
age = int(age)  # 將字串轉換為整數
print("你的年齡是：", age)

範例：多次輸入

我們可以利用 input() 函數多次取得使用者輸入：

name = input("請輸入你的名字：")
age = int(input("請輸入你的年齡："))
print("你好,", name, "! 你今年", age, "歲。")

常見問題與解答

問題 1：為什麼我的數字運算會出錯？

因為 input() 的輸入預設為字串，直接對字串進行運算會導致錯誤。請確保先將輸入轉換為適當的型態。

問題 2：如何在提示訊息後加入空格？

可以在提示訊息的字串尾部加入空格，例如 input("請輸入你的名字： ")。

問題 3：可以一次取得多個輸入嗎？

可以使用 split() 方法，將使用者輸入的字串拆分為多個部分：

data = input("請輸入你的名字和年齡，用空格分隔：").split()
name = data[0]
age = int(data[1])
print("你好,", name, "! 你今年", age, "歲。")

挑戰題

挑戰 1：簡單的問答程式 🛈

建立一個程式，詢問使用者的名字與興趣，然後輸出一段包含這些資訊的訊息。

# 解答
name = input("請輸入你的名字：")
hobby = input("請輸入你的興趣：")
print("你好,", name, "! 你的興趣是", hobby, "。")

挑戰 2：計算 BMI 🛈

建立一個程式，要求使用者輸入身高（公分）和體重（公斤），計算並輸出 BMI 值。

# 解答
height = float(input("請輸入你的身高（公分）："))
weight = float(input("請輸入你的體重（公斤）："))
bmi = weight / (height/100 * height/100)
print("你的身高是", height, "公分，體重是", weight, "公斤，BMI 值是", round(bmi, 2))

挑戰 3：簡易計算機 🛈

建立一個程式，要求使用者輸入兩個數字，然後進行加法、減法、乘法及除法運算，並輸出結果。

# 解答
num1 = float(input("請輸入第一個數字："))
num2 = float(input("請輸入第二個數字："))

print("加法結果：", num1 + num2)
print("減法結果：", num1 - num2)
print("乘法結果：", num1 * num2)
print("除法結果：", num1 / num2)

挑戰 4：檢查輸入是否為數字 🛈

建立一個程式，要求使用者輸入一個值，檢查該值是否為數字。如果是數字則輸出「這是一個數字」，否則輸出「這不是數字」。

# 解答
value = input("請輸入一個值：")
if value.isdigit():
    print("這是一個數字")
else:
    print("這不是數字")

挑戰 5：計算未來的年齡 🛈

建立一個程式，要求使用者輸入目前的年齡，然後計算並輸出 5 年後的年齡。

# 解答
current_age = int(input("請輸入你的目前年齡："))
future_age = current_age + 5
print("5 年後你的年齡將是", future_age)

挑戰 6：字串操作 🛈

建立一個程式，要求使用者輸入一段文字，並輸出該文字的長度以及全大寫版本。

# 解答
text = input("請輸入一段文字：")
print("文字長度為", len(text))
print("全大寫文字為", text.upper())

挑戰 7：分數平均計算 🛈

建立一個程式，要求使用者輸入三個分數，然後計算並輸出它們的平均值。

# 解答
score1 = float(input("請輸入第一個分數："))
score2 = float(input("請輸入第二個分數："))
score3 = float(input("請輸入第三個分數："))
average = (score1 + score2 + score3) / 3
print("平均分數是：", average)

挑戰 8：使用分隔符號處理多輸入 🛈

建立一個程式，要求使用者輸入三個以逗號分隔的數字，分別儲存到三個變數，然後輸出它們的總和。

# 解答
numbers = input("請輸入三個以逗號分隔的數字：").split(",")
num1 = float(numbers[0])
num2 = float(numbers[1])
num3 = float(numbers[2])
total = num1 + num2 + num3
print("總和是：", total)
