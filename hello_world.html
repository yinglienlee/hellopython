什麼是 Python？

Python 是一種高階、直譯式程式設計語言，由 Guido van Rossum 在 1991 年發表。其設計理念是簡單、易讀和高效率，非常適合初學者以及專業開發者。

Python 的應用非常廣泛，涵蓋以下領域：

  • 資料分析與科學計算：支援工具如 Pandas、NumPy 和 Matplotlib。
  • 人工智慧與機器學習：廣泛使用 TensorFlow、PyTorch 等框架。
  • 網頁開發：使用 Django 和 Flask 開發動態網站。
  • 自動化任務：撰寫腳本自動完成重複性工作。
  • 遊戲開發：使用 Pygame 等模組開發簡單遊戲。

為什麼學習 Python 是個好的選擇？

以下是 Python 的主要優勢，使其成為學習程式設計的理想選擇：

  • 易於學習： Python 語法簡潔直觀，接近自然語言，非常適合初學者快速上手。
  • 跨平台支援： 無論是 Windows、Mac 還是 Linux，Python 都能執行 Python 程式。
  • 社群支持： Python 擁有龐大的開發者社群，提供豐富的學習資源和第三方模組。
  • 多功能性： 從資料科學到網頁開發，Python 幾乎可以應用於任何領域。
  • 快速開發： 使用 Python 可以以更少的程式碼完成複雜的任務，提高生產力。

無論你是對資料分析、人工智慧還是網頁開發感興趣，Python 都能幫助你實現目標。

什麼是Thonny？

Thonny 是一款專為初學者設計的Python編輯器。它簡單易用，但同時擁有許多強大的功能，可以幫助你快速上手程式設計。

如果你還沒有安裝Thonny，請按照以下步驟操作：

  1. 到 Thonny 官方網站 下載適合你作業系統的版本。
  2. 依照指示安裝Thonny。
  3. 打開Thonny，你會看到一個簡潔的介面，準備好迎接你的第一行程式碼！

撰寫第一個Python程式

我們的目標是讓電腦顯示出經典的 "Hello, World!" 字串，這是許多程式設計語言的傳統起點。

步驟 1：建立新檔案

  1. 在Thonny中，點擊左上角的 File 選單，選擇 New。
  2. 一個空白的編輯器視窗將會出現，這就是你撰寫程式碼的地方。

步驟 2：輸入程式碼

在空白的編輯器視窗中，輸入以下程式碼：

print("Hello, World!")

現在，讓我們來分解這行程式碼：

  • print：這是一個Python的內建函式，用來在螢幕上顯示文字或其他內容。
  • ("Hello, World!")：括號中的內容是我們想顯示的字串。字串需要用雙引號（"）或單引號（'）包裹起來。
  • "Hello, World!"：這是我們的訊息，會被顯示在螢幕上。

步驟 3：儲存檔案

  1. 點擊左上角的 File 選單，選擇 Save As。
  2. 選擇一個資料夾，並為你的檔案取一個名稱，例如 hello_world.py。
  3. 點擊 Save 按鈕。

步驟 4：執行程式

  1. 在Thonny中，點擊編輯器上方的 Run 按鈕（綠色三角形）。
  2. 或者，按下鍵盤上的 F5 快捷鍵。
  3. 程式將執行，你會在下方的 Shell 視窗中看到：
Hello, World!

恭喜！你已經成功撰寫並執行了你的第一個Python程式。

使用註解

在程式碼中加入註解是非常重要的，它可以幫助你記錄程式邏輯，並方便其他人理解你的程式。

Python 的註解有以下幾種用法：

  • 單行註解：使用井字號（#）來標註，例如：
# 這是一個註解，Python 不會執行這行文字
print("Hello, World!")  # 這是另一個註解
  • 多行註解：雖然 Python 沒有專門的多行註解語法，但可以使用多行字串來模擬，例如：
"""
這是一個多行註解
適合用來說明程式碼的邏輯
或其他詳細資訊
"""
print("Hello, World!")

記得善用註解來提升程式的可讀性，特別是在撰寫複雜的程式時！

常見問題與解答

問題 1：執行程式時，沒有看到任何輸出？

  • 確保你已儲存檔案，並且點擊了 Run 按鈕。
  • 確保你的程式碼完全正確，包括引號與括號。

問題 2：輸出錯誤訊息 "SyntaxError: unterminated string literal (detected at line 1)"？

這通常是因為字串沒有正確關閉。例如：

print("Hello, World!)  # 缺少結束的引號

確認每個字串都用配對的引號包起來。

問題 3：輸出錯誤訊息 "IndentationError: unexpected indent"？

這是因為程式碼的縮排不正確。例如：

 print("Hello, World!")  # 不應該多一個空格

確保程式碼的每一行左邊都沒有多餘的空格。

問題 4：執行程式後出現 "NameError: name 'pritn' is not defined"？

這是因為打錯了函式名稱，例如：

pritn("Hello, World!")  # 應該是 print

確認函式名稱正確無誤。

問題 5：輸出錯誤訊息 "TypeError: 'str' object is not callable"？

這是因為不小心將內建函式名稱重新定義為字串，例如：

print = "Hello"  # 將內建函式覆蓋
print("World")  # 會出錯

避免重新定義內建函式名稱。

挑戰 8：使用反斜線讓程式碼換行 🛈

# 使用反斜線讓程式碼換行
result = 10 * 2 + \
         30 * 3 + \
         40 * 4 + \
         50 * 5 + \
         60 * 6 + \
         70 * 7

print("計算結果:", result)
