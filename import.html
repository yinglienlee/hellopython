模組的基本用法

1. 什麼是模組？

模組是一個包含 Python 程式碼的檔案，其副檔名為 .py。透過匯入模組，我們可以使用其定義的函式、類別和變數。

2. 匯入內建模組

Python 提供了許多內建模組，例如 math 和 random：

import math

# 使用 math 模組的函式
print(math.sqrt(16))  # 輸出：4.0

3. 匯入外部模組

外部模組通常需要透過 pip 安裝，例如 requests：

# 安裝 requests 模組
# pip install requests

import requests

response = requests.get("https://www.example.com")

if response.status_code == 200:
    html = response.text
    print(html)
else:
    print(f"Failed to retrieve page. Status code: {response.status_code}")

Thonny 也提供便捷的安裝方式

  1. 打開 Thonny 開發環境。
  2. 點擊選單中的 工具 (Tools)，然後選擇 管理套件 (Manage Packages)。
  3. 在彈出的視窗中，於搜索框輸入 requests。
  4. 點擊 安裝 (Install) 按鈕，等待安裝完成。

4. 匯入自訂模組

我們可以匯入自己定義的模組。例如，有一個名為 mymodule.py 的檔案：

# mymodule.py
def greet(name):
    return f"Hello, {name}!"

# 匯入自訂模組
import mymodule

print(mymodule.greet("Alice"))

5. 使用 from ... import ...

可以只匯入模組中的特定部分：

from math import pi, sqrt

print(pi)  # 輸出：3.141592653589793
print(sqrt(16))  # 輸出：4.0

6. 模組別名（Alias）

有時候模組名稱較長或希望使用簡潔的名稱，可以使用 as 關鍵字來為模組取別名（alias）。這對於簡化程式碼或解決名稱衝突非常有用。

import math as m

# 使用模組的別名 m
print(m.sqrt(16))  # 輸出：4.0

常見問題與解答

問題 1：匯入模組時為什麼會出錯？

確保模組已安裝且位於 Python 的模組路徑中。

問題 2：如何檢查安裝的外部模組？

可以在命令提示字元中使用 pip list 查看已安裝的模組。在 Thonny 中則可點擊選單中的 工具 (Tools)，然後選擇 Open system shell 開啟 Thonny 環境的命令提示字元，再執行 pip list。

問題 3：匯入自訂模組時為什麼找不到檔案？

以下方法都可以確保 Python 能正確地找到並匯入你的自訂模組。

  • 同一目錄： 將自訂模組與執行的程式放在同一個資料夾內。
  • 修改 sys.path： 在程式中使用 sys.path.append() 將模組路徑加入搜尋清單。
  • 設定 PYTHONPATH： 在作業系統中設定環境變數，讓 Python 自動搜尋指定路徑。

挑戰題目

挑戰 1：計算圓的面積 🛈

撰寫一個程式，匯入 math 模組，要求使用者輸入半徑，並計算圓的面積。

# 範例輸入
# 半徑：5
# 預期輸出
圓的面積為：78.54
# 解答
import math

# 要求使用者輸入半徑
radius = float(input("請輸入圓的半徑："))
# 計算圓的面積
area = math.pi * (radius ** 2)
print(f"圓的面積為：{area:.2f}")

挑戰 2：下載網頁內容 🛈

使用 requests 模組，建立一個程式，下載並顯示特定網址的內容。

# 範例輸入
# 網址：https://www.example.com
# 預期輸出
網站內容：


  ...

# 解答
import requests

# 輸入網址
url = input("請輸入網址：")
# 發送請求並獲取響應
response = requests.get(url)
# 顯示網站內容
print("網站內容：")
print(response.text)

挑戰 3：建立與匯入自訂模組 🛈

建立一個包含函式的自訂模組，該函式接收數字串列並回傳其平均值，然後在主程式中匯入並使用此模組。

# 範例輸入
# 自訂模組：utils/math_utils.py
def calculate_average(numbers):
    return sum(numbers) / len(numbers)

# 主程式
from utils.math_utils import calculate_average

numbers = [10, 20, 30]
print(f"平均值：{calculate_average(numbers)}")
# 解答
# utils/math_utils.py
def calculate_average(numbers):
    return sum(numbers) / len(numbers)

# 主程式 main.py
from utils.math_utils import calculate_average

# 輸入數字串列
numbers = [float(num) for num in input("請輸入數字串列（以逗號分隔）：").split(",")]
# 計算並輸出平均值
print(f"平均值：{calculate_average(numbers):.2f}")

挑戰 4：匯入 datetime 模組 🛈

設計一個程式，匯入 datetime 模組，並輸出當前的日期與時間。

# 範例輸入
# 匯入 datetime 模組
from datetime import datetime

# 預期輸出
當前日期與時間：2025-01-20 10:30:45
# 解答
from datetime import datetime

# 獲取當前日期與時間
current_datetime = datetime.now()
# 輸出結果
print(f"當前日期與時間：{current_datetime.strftime('%Y-%m-%d %H:%M:%S')}")

挑戰 5：匯入與重命名模組 🛈

匯入 math 模組並使用別名，例如 m，計算兩個數字的平方根之和。

# 範例輸入
數字 9 和 16
# 預期輸出
平方根之和為：7.0
# 解答
import math as m

# 定義數字
num1, num2 = 9, 16
# 計算平方根之和
result = m.sqrt(num1) + m.sqrt(num2)
# 輸出結果
print(f"平方根之和為：{result}")

# 預期輸出
requests 模組版本：2.26.0

挑戰 7：使用 os 模組操作檔案 🛈

設計一個程式，匯入 os 模組，並列出當前目錄中的所有檔案與資料夾。

# 預期輸出
當前目錄內容：['file1.txt', 'file2.txt', 'folder1']
# 解答
import os

# 獲取當前目錄內容
contents = os.listdir(".")
# 輸出結果
print(f"當前目錄內容：{contents}")

挑戰 8：建立與匯入多層模組 🛈

建立多層目錄結構，匯入模組並使用其中的函式。例如，建立 utils/math_utils.py，實現一個加法函式。

# 範例輸入
# utils/math_utils.py
def add(a, b):
    return a + b

# 主程式
from utils.math_utils import add

# 預期輸出
加法結果：7
# 解答
# utils/math_utils.py
def add(a, b):
    return a + b

# 主程式 main.py
from utils.math_utils import add

# 使用加法函式
result = add(3, 4)
print(f"加法結果：{result}")
