自訂函式

學習導覽

本章節將介紹如何定義與使用 Python 自訂函式。函式是程式設計的基礎單位，能夠提升程式碼的重用性與可讀性。

自訂函式的基本概念

1. 什麼是函式？

函式是一段具名的程式碼區塊，可以接收輸入（參數），執行特定任務，並選擇性地回傳輸出（回傳值）。

2. 定義與呼叫函式

# 定義函式
def greet(name):
    print(f"你好, {name}!")

# 呼叫函式
greet("Alice")  # 輸出：你好, Alice!

3. 函式的參數與回傳值

# 帶有回傳值的函式
def add(a, b):
    return a + b

result = add(3, 5)
print(result)  # 輸出：8

4. 匿名函式 (Lambda)

匿名函式是沒有名稱的簡單函式，使用 lambda 關鍵字定義。

# 使用 lambda 定義函式
square = lambda x: x ** 2
print(square(4))  # 輸出：16

5. 使用 Lambda 作為排序鍵

Lambda 常用於傳遞簡單的函式作為其他函式的參數，例如用於排序列表中的元素。

words = ["apple", "banana", "cherry", "date"]
words.sort(key=lambda word: len(word))
print(words)  # 輸出：['date', 'apple', 'banana', 'cherry']

解釋：

  • lambda word: len(word) 是一個匿名函式，接收一個字串並返回其長度。
  • sort 方法使用這個匿名函式來決定排序順序，因此列表按照字串長度從小到大排序。

這種用法在需要簡單且一次性函式的情況下非常實用，例如排序、篩選或轉換資料時。

常見問題與解答

問題 1：函式可以沒有回傳值嗎？

可以，若函式未指定回傳值，則預設返回 None。

問題 2：如何設定函式參數的預設值？

在參數名稱後面指定預設值，例如 def greet(name="訪客")。

問題 3：可以在函式內呼叫其他函式嗎？

可以，這是模組化程式設計的常見做法。

挑戰題目

挑戰 1：計算矩形面積 🛈

撰寫一個函式，接收長與寬作為參數，並回傳矩形的面積。

# 解答
def rectangle_area(length, width):
    return length * width

print(rectangle_area(5, 10))  # 輸出：50

挑戰 2：溫度轉換 🛈

設計一個函式，將攝氏溫度轉換為華氏溫度，並回傳結果。

# 解答
def celsius_to_fahrenheit(celsius):
    return celsius * 9/5 + 32

print(celsius_to_fahrenheit(25))  # 輸出：77.0

挑戰 3：應用 Lambda 函式 🛈

使用 lambda 函式，實現一個功能，接收圓的半徑，並返回圓的面積。

# 解答
circle_area = lambda radius: 3.14159 * radius ** 2
print(circle_area(5))  # 輸出：78.53975

挑戰 4：應用 Lambda 函式 🛈

給定一組數字 numbers = [5, -10, 3, -1, 7]，使用 sorted 函式和 lambda，將數字按照它們的絕對值進行排序，並輸出排序結果。

# 解答
numbers = [5, -10, 3, -1, 7]
sorted_numbers = sorted(numbers, key=lambda x: abs(x))
print(sorted_numbers)  # 輸出：[3, -1, 5, 7, -10]

挑戰 5：判斷閏年 🛈

撰寫一個函式，接收一個年份作為參數，並回傳該年份是否為閏年。若未提供年份，則預設為 2025。

# 解答
def is_leap_year(year=2025):
    return (year % 4 == 0 and year % 100 != 0) or (year % 400 == 0)

print(is_leap_year(2024))  # 輸出：True

挑戰 6：清單中的最大值 🛈

撰寫一個函式，接收一個數字清單，並回傳該清單中的最大值。

# 解答
def max_in_list(numbers):
    return max(numbers)

print(max_in_list([10, 20, 5, 30]))  # 輸出：30

挑戰 7：使用 Lambda 過濾清單 🛈

給定一個數字清單 numbers = [10, -5, 15, -20, 30]，使用 filter 和 lambda，篩選出清單中所有大於 0 的數字，並輸出結果。

# 解答
numbers = [10, -5, 15, -20, 30]
positive_numbers = list(filter(lambda x: x > 0, numbers))
print(positive_numbers)  # 輸出：[10, 15, 30]

挑戰 8：計算多個矩形的面積 🛈

撰寫一個函式，接收多個矩形的長與寬（作為一個清單），並回傳所有矩形面積的總和。

# 解答
def total_area(rectangles):
    return sum(length * width for length, width in rectangles)

print(total_area([(5, 10), (3, 4), (2, 6)]))  # 輸出：62

挑戰 9：應用 Lambda 進行資料轉換 🛈

設計一段程式碼，使用 map 和 lambda 將一組攝氏溫度清單 [0, 20, 37, 100] 轉換為華氏溫度，並輸出結果。

# 解答
celsius_list = [0, 20, 37, 100]
fahrenheit_list = list(map(lambda c: c * 9/5 + 32, celsius_list))
print(fahrenheit_list)  # 輸出：[32.0, 68.0, 98.6, 212.0]

結語

自訂函式是 Python 中提升程式碼可讀性與重用性的關鍵技術。透過學習與實作，你將能更靈活地運用函式來解決各種問題。
