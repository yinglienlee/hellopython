迴圈

學習導覽

迴圈是程式設計中的重要概念，它能讓我們重複執行一段程式碼，直到達成特定條件為止。透過迴圈，我們可以讓程式更簡潔。本章將涵蓋：

  • for 迴圈的基本使用。
  • while 迴圈的基本使用。
  • 理解迴圈控制語法：break 和 continue。
  • 巢狀迴圈的運用。

迴圈語法與使用

1. 認識 range() 函數

range() 是 Python 中生成一系列數字的內建函數，常與 for 迴圈一起使用。基本語法為：

range(start, stop, step)
  • start（可選）：起始值，預設為 0。
  • stop（必填）：結束值（不包含在範圍內）。
  • step（可選）：步進值，預設為 1。

以下是一些範例：

# 範例 1：生成 0 到 4
for i in range(5):
    print(i)  # 輸出 0, 1, 2, 3, 4

# 範例 2：指定起始值
for i in range(2, 6):
    print(i)  # 輸出 2, 3, 4, 5

# 範例 3：指定步進值
for i in range(0, 10, 2):
    print(i)  # 輸出 0, 2, 4, 6, 8

# 範例 4：反向生成
for i in range(10, 0, -2):
    print(i)  # 輸出 10, 8, 6, 4, 2

range() 函數非常靈活，能幫助你輕鬆處理不同範圍的數字迭代。

2. for 迴圈

for 迴圈用於依序遍歷可迭代物件（例如串列或字串）：

這段程式碼將輸出從 0 到 4 的數字。

for i in range(5):
    print(i)

輸出結果：
0
1
2
3
4

這段程式碼將輸出字串中各個字元。

for c in "Formosa":
    print(c)

輸出結果：
F
o
r
m
o
s
a

3. while 迴圈

while 迴圈用於在條件成立時持續執行程式碼：

count = 0
while count < 5:
    print(count)
    count += 1

輸出結果與 for 迴圈相同：
0
1
2
3
4

4. break 與 continue

break 用於中止迴圈，continue 則用於跳過當前迭代並進入下一次迭代：

# 使用 break
for i in range(10):
    if i == 5:
        break
    print(i)

輸出結果：
0
1
2
3
4

# 使用 continue
for i in range(5):
    if i == 2:
        continue
    print(i)

輸出結果：
0
1
3
4

5. 巢狀迴圈

巢狀迴圈是指一個迴圈內部嵌套另一個迴圈：

for i in range(3):
    for j in range(2):
        print(f"i={i}, j={j}")

輸出結果：
i=0, j=0
i=0, j=1
i=1, j=0
i=1, j=1
i=2, j=0
i=2, j=1

常見問題與解答

問題 1：如何避免 while 迴圈無限執行？

確保迴圈條件會在某個時候變為 False，例如：

count = 0
while count < 5:
    print(count)
    count += 1  # 確保條件會改變

問題 2：如何遍歷串列中的每個元素？

使用 for 迴圈：

fruits = ["apple", "banana", "cherry"]
for fruit in fruits:
    print(fruit)

輸出結果：
apple
banana
cherry

問題 3：如何在巢狀迴圈中結束所有迴圈？

當我們在巢狀迴圈（Nested Loops）中，想要從內部迴圈跳出並終止所有外部迴圈， break 只能終止當前的迴圈，無法直接影響外部的迴圈。因此，我們需要使用旗標變數（flag variable）來標記某個條件是否達成，然後在外部迴圈判斷該變數來決定是否跳出。

found = False
for i in range(3):
    for j in range(3):
        if i == 1 and j == 1:
            found = True
            break
    if found:
        break
print("結束所有迴圈")

挑戰題目

挑戰 1：計算總和 🛈

寫一段程式碼，使用 for 迴圈計算從 1 到 100 的總和。

# 解答
total = 0
for i in range(1, 101):
    total += i
print(f"1 到 100 的總和是：{total}")

挑戰 2：九九乘法表 🛈

使用巢狀迴圈輸出完整的九九乘法表。

# 解答
for i in range(1, 10):
    for j in range(1, 10):
        print(f"{i} x {j} = {i * j}", end="\t")
    print()  # 換行

挑戰 3：尋找質數 🛈

設計一個程式，找出 1 到 50 範圍內的所有質數。

# 解答
for num in range(2, 51):
    is_prime = True
    for i in range(2, int(num ** 0.5) + 1):
        if num % i == 0:
            is_prime = False
            break
    if is_prime:
        print(num, end=" ")

挑戰 4：反向輸出 🛈

設計一個程式，使用 for 迴圈，輸出 10 到 1 的數字（反向遞減）。

# 解答
for i in range(10, 0, -1):
    print(i, end=" ")

挑戰 5：計算階乘 🛈

撰寫一段程式碼，讓使用者輸入一個正整數，並使用 while 迴圈計算該數字的階乘（例如 5! = 5 × 4 × 3 × 2 × 1）。

# 解答
num = int(input("請輸入一個正整數："))
factorial = 1
while num > 0:
    factorial *= num
    num -= 1
print(f"階乘結果是：{factorial}")

挑戰 7：質數檢查 🛈

設計一個程式，讓使用者輸入一個正整數，檢查該數字是否為質數。

# 解答
num = int(input("請輸入一個正整數："))
if num < 2:
    print("不是質數")
else:
    is_prime = True
    for i in range(2, int(num ** 0.5) + 1):
        if num % i == 0:
            is_prime = False
            break
    if is_prime:
        print(f"{num} 是質數")
    else:
        print(f"{num} 不是質數")

挑戰 8：數字猜猜樂 🛈

撰寫一個程式，讓電腦隨機生成一個 1 到 100 之間的數字，並讓使用者猜測，程式應提示使用者猜得太高或太低，直到猜對為止。

# 解答
import random

target = random.randint(1, 100)
guess = 0
while guess != target:
    guess = int(input("請猜一個 1 到 100 的數字："))
    if guess < target:
        print("太低了！")
    elif guess > target:
        print("太高了！")
    else:
        print("恭喜你，猜對了！")
