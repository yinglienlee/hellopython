字典容器

學習導覽

在這一章中，我們將學習字典 (dictionary)，這是一種用於儲存鍵值組 (key-value pairs) 的資料結構，適合快速查詢與組織資料。

  • 了解如何建立字典。
  • 學習如何存取與修改字典中的值。
  • 探索字典的常用方法。
  • 學習如何迭代字典。

字典的基本操作

1. 建立字典

字典使用大括號 {} 表示，鍵和值之間使用冒號分隔：

# 建立空字典
empty_dict = {}

# 建立包含鍵值組的字典
person = {
    "name": "Alice",
    "age": 25,
    "city": "Taipei"
}

2. 存取與修改值

使用鍵存取字典中的值，並可以直接修改：

# 存取值
print(person["name"])  # 輸出：Alice

# 修改值
person["age"] = 26
print(person)  # 輸出：{'name': 'Alice', 'age': 26, 'city': 'Taipei'}

3. 新增與刪除鍵值組

可以直接新增或使用 del 刪除鍵值組：

# 新增鍵值組
person["job"] = "Engineer"
print(person)  # 輸出：{'name': 'Alice', 'age': 26, 'city': 'Taipei', 'job': 'Engineer'}

# 刪除鍵值組
del person["city"]
print(person)  # 輸出：{'name': 'Alice', 'age': 26, 'job': 'Engineer'}

4. 字典的常用方法

以下是一些常見的方法：

# 獲取所有鍵
print(person.keys())  # 輸出：dict_keys(['name', 'age', 'job'])

# 獲取所有值
print(person.values())  # 輸出：dict_values(['Alice', 26, 'Engineer'])

# 獲取所有鍵值組
print(person.items())  # 輸出：dict_items([('name', 'Alice'), ('age', 26), ('job', 'Engineer')])

5. 學習如何迭代字典

有時候，我們會想要逐一處理字典裡的每一筆資料。這時候就需要使用「迭代（iteration）」的方法。你可以使用 for 迴圈來達成這個目標。

5.1. 列出所有的 key

my_dict = {'apple': 3, 'banana': 5, 'orange': 2}
for key in my_dict:
    print(key, my_dict[key], sep=": ")

輸出：

apple
banana
orange

5.2. 列出所有的 value

my_dict = {'apple': 3, 'banana': 5, 'orange': 2}
for value in my_dict.values():
    print(value)

輸出：

3
5
2

5.3. 同時列出 key 和 value

my_dict = {'apple': 3, 'banana': 5, 'orange': 2}
for key, value in my_dict.items():
    print(f"{key}: {value}")

輸出：

apple: 3
banana: 5
orange: 2

常見問題與解答

問題 1：字典的鍵有什麼限制？

字典的鍵必須是不可變的，例如數字、字串或元組，但不能是串列或集合。

問題 2：如何避免存取不存在的鍵時發生錯誤？

可以使用 get() 方法，指定預設值：

print(person.get("country", "Unknown"))  # 輸出：Unknown

問題 3：如何合併兩個字典？

在 Python 3.9+ 中可以使用 | 運算符：

# 合併字典
dict1 = {"a": 1, "b": 2}
dict2 = {"b": 3, "c": 4}
combined = dict1 | dict2
print(combined)  # 輸出：{'a': 1, 'b': 3, 'c': 4}

挑戰題目

挑戰 1：學生成績管理 🛈

設計一個程式，讓使用者輸入多位學生的名字與成績，儲存於字典中，並輸出所有學生的平均成績。

# 解答
students = {}
while True:
    name = input("請輸入學生姓名（輸入 '結束' 結束）：")
    if name == "結束":
        break
    score = int(input(f"請輸入{name}的成績："))
    students[name] = score

average_score = sum(students.values()) / len(students)
print("學生平均成績：", average_score)

挑戰 2：單字翻譯 🛈

建立一個字典儲存英文單字與中文翻譯，讓使用者輸入英文單字並查詢對應的中文翻譯。

# 解答
dictionary = {"apple": "蘋果", "banana": "香蕉", "orange": "橘子"}
word = input("請輸入英文單字：")
translation = dictionary.get(word, "該單字不存在於字典中")
print(f"{word} 的翻譯是：{translation}")

挑戰 3：購物清單 🛈

設計一個購物清單程式，讓使用者新增商品與數量，並列出完整的購物清單與總數量。

# 解答
shopping_list = {}
while True:
    item = input("請輸入商品名稱（輸入 '結束' 結束）：")
    if item == "結束":
        break
    quantity = int(input(f"請輸入{item}的數量："))
    shopping_list[item] = shopping_list.get(item, 0) + quantity

total_quantity = sum(shopping_list.values())
print("購物清單：", shopping_list)
print("總數量：", total_quantity)

挑戰 4：熱門單字統計 🛈

給定一段文字，計算其中每個單字出現的次數，並使用字典儲存結果。

# 範例輸入
text = "apple banana apple orange apple banana"
# 預期輸出
{'apple': 3, 'banana': 2, 'orange': 1}
# 解答
text = "apple banana apple orange apple banana"
words = text.split()
word_count = {}
for word in words:
    word_count[word] = word_count.get(word, 0) + 1
print(word_count)

挑戰 5：電話簿管理 🛈

設計一個程式，允許使用者新增聯絡人（姓名與電話），查詢聯絡人電話，並列出所有聯絡人。

# 範例輸入
add_contact("Alice", "12345678")
get_contact("Alice")  # 輸出：12345678
list_contacts()        # 輸出：{'Alice': '12345678'}
# 解答
phone_book = {}

def add_contact(name, phone):
    phone_book[name] = phone
    print(f"已新增聯絡人：{name}")

def get_contact(name):
    print(phone_book.get(name, "聯絡人不存在"))

def list_contacts():
    print("電話簿：", phone_book)

add_contact("Alice", "12345678")
get_contact("Alice")
list_contacts()

挑戰 6：反轉鍵值 🛈

給定一個字典，將其鍵與值進行交換，並輸出結果。

# 範例輸入
original = {"a": 1, "b": 2, "c": 3}
# 預期輸出
{1: 'a', 2: 'b', 3: 'c'}
# 解答
original = {"a": 1, "b": 2, "c": 3}
reversed_dict = {v: k for k, v in original.items()}
print(reversed_dict)

挑戰 7：成績排名 🛈

給定一個包含學生名字與成績的字典，將學生按成績從高到低排序並輸出結果。

# 範例輸入
grades = {"Alice": 85, "Bob": 92, "Charlie": 88}
# 預期輸出
[('Bob', 92), ('Charlie', 88), ('Alice', 85)]
# 解答
grades = {"Alice": 85, "Bob": 92, "Charlie": 88}
sorted_grades = sorted(grades.items(), key=lambda x: x[1], reverse=True)
print(sorted_grades)

挑戰 8：合併字典 🛈

給定兩個字典，將它們合併，若有重複的鍵則保留較大的值。

# 範例輸入
dict1 = {"a": 1, "b": 2}
dict2 = {"b": 3, "c": 4}
# 預期輸出
{'a': 1, 'b': 3, 'c': 4}
# 解答
dict1 = {"a": 1, "b": 2}
dict2 = {"b": 3, "c": 4}
merged = {k: max(dict1.get(k, 0), dict2.get(k, 0)) for k in dict1.keys() | dict2.keys()}
print(merged)
