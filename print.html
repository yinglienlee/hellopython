print 進階

學習導覽

本章將帶你深入了解 print() 函式的進階用法，不僅限於基本的文字輸出，還包括如何格式化輸出、控制換行、以及使用自訂分隔符號等技巧。掌握這些技能，能讓你的程式更具表現力！

  • 學會使用 sep 和 end 參數。
  • 了解如何使用特殊字元（如換行符號和定位符號）。
  • 掌握格式化輸出的方法，讓輸出內容更加美觀。

探索 print 函式的進階功能

使用 end 控制行結尾

預設情況下，print() 會在每次輸出後自動換行。但你可以使用 end 參數自訂行結尾，例如：

print("Hello", end=" ")
print("World!")

輸出結果：
Hello World!

使用 sep 自訂分隔符號

當你想一次輸出多個值時，可以用 sep 參數自訂它們之間的分隔符號：

print("Python", "is", "fun", sep=" - ")

輸出結果：
Python - is - fun

加入特殊字元 - Escape Sequence

你可以在輸出中使用一種稱為 Escape Sequence 的特殊字元來控制格式，例如換行符號 \n 和定位符號 \t：

print("第一行\n第二行")
print("A\tB\tC")

輸出結果：
第一行
第二行
A    B    C

了解 Escape Sequence

Escape sequence 是一組以反斜線（\）開頭的特殊字符，用於在字串中表示一些特殊的動作或字元，例如換行、定位或插入引號。這些對於格式化輸出非常重要。

常見的 Escape Sequence

  • \n：換行符號
  • \t：定位符號
  • \\：插入反斜線
  • \'：插入單引號
  • \"：插入雙引號

範例

以下是一些常見的 Escape Sequence 用法範例：

print("第一行\n第二行") # 使用換行符號
print("A\tB\tC")     # 使用定位符號
print("She said, \"Hello!\"") # 插入雙引號

輸出結果：

第一行
第二行
A       B       C
She said, "Hello!"

常見問題與解答

問題 1：如何避免輸出後自動換行？

使用 end 參數。例如：

print("這裡", end=" ")
print("不換行")

輸出結果：
這裡 不換行

問題 2：如何在輸出中加入換行或定位符號？

使用特殊字元，例如：

print("第一行\n第二行")
print("A\tB\tC")

問題 3：輸出多個值時，能否自訂分隔符號？

可以使用 sep 參數，例如：

print("Apple", "Banana", "Cherry", sep=" | ")

挑戰題目

挑戰 1：使用 print() 繪製一個矩形 🛈

輸出如下圖案：

****
****
****
****
# 解答
for _ in range(4):
    print("****")

挑戰 2：創建進度條 🛈

讓程式輸出類似以下的進度條：

1/5 -> 2/5 -> 3/5 -> 4/5 -> 5/5 -> 完成!
# 解答
for i in range(1, 6):
    print(f"{i}/5", end=" -> ")
print("完成!")

挑戰 3：格式化日期輸出 🛈

設計程式輸出以下格式的日期：
今天是：2025/01/04

# 解答
from datetime import date
today = date.today()
year = today.year
month = today.month
day = today.day
print("今天是：", year, "/", month, "/", day, sep="")

挑戰 4：多行輸出 🛈

設計程式輸出以下格式的內容，使用 \n 實現換行：

Hello
Python
World
# 解答
print("Hello\nPython\nWorld")

挑戰 5：自訂分隔符號 🛈

讓程式輸出以下內容，使用 sep 參數自訂分隔符號：

Apple | Banana | Cherry
# 解答
print("Apple", "Banana", "Cherry", sep=" | ")

挑戰 6：使用定位符號格式化輸出 🛈

設計程式輸出以下對齊的內容，使用 \t 格式化：

Name	Age	City
Alice	25	New York
Bob	30	San Francisco
# 解答
print("Name\tAge\tCity")
print("Alice\t25\tNew York")
print("Bob\t30\tSan Francisco")

挑戰 7：顯示引號 🛈

讓程式輸出以下內容，使用 Escape Sequence 插入雙引號：

She said, "Hello, World!"
# 解答
print("She said, \"Hello, World!\"")

挑戰 8：綜合使用 end 和 sep

讓程式輸出以下內容，使用 sep 設置分隔符號，並用 end 控制行尾：

Apple - Banana - Cherry !
# 解答
print("Apple", "Banana", "Cherry", sep=" - ", end=" !")
