九九乘法表

題目說明

撰寫一個程式，利用巢狀迴圈印出 1 到 9 的乘法表，每一項以「a x b = c」呈現，並利用格式化字串確保表格對齊。

參考解答

# 九九乘法表
for i in range(1, 10):
    for j in range(1, 10):
        # 使用 f-string 格式化，每項以 tab 分隔
        print(f"{i}x{j}={i*j}", end="\t")
    print()  # 換行

解說

外層迴圈控制乘數從 1 到 9，內層迴圈控制被乘數從 1 到 9，每次計算 i 與 j 的乘積，並以格式化輸出，項與項之間以 Tab 隔開。

溫度轉換器

題目說明

設計一個程式，讓使用者選擇轉換模式：

  • 1. 攝氏轉華氏
  • 2. 華氏轉攝氏

依照使用者的選擇，再提示輸入相應的溫度，並利用公式進行轉換：

  • 攝氏轉華氏： F = C × 9/5 + 32
  • 華氏轉攝氏： C = (F − 32) × 5/9

參考解答

def celsius_to_fahrenheit(c):
    """攝氏轉華氏"""
    return c * 9/5 + 32

def fahrenheit_to_celsius(f):
    """華氏轉攝氏"""
    return (f - 32) * 5/9

print("溫度轉換器")
print("選擇轉換模式:")
print("1. 攝氏轉華氏")
print("2. 華氏轉攝氏")

mode = input("請輸入選項 (1/2): ")

if mode == "1":
    c = float(input("請輸入攝氏溫度: "))
    f = celsius_to_fahrenheit(c)
    print(f"{c}°C 轉換為華氏溫度是 {f:.2f}°F")
elif mode == "2":
    f = float(input("請輸入華氏溫度: "))
    c = fahrenheit_to_celsius(f)
    print(f"{f}°F 轉換為攝氏溫度是 {c:.2f}°C")
else:
    print("輸入錯誤，請重新執行程式並輸入正確選項。")

解說

本程式首先定義了兩個函式，分別用於攝氏與華氏的轉換，然後根據使用者的選擇提示輸入相應溫度，最後利用格式化輸出轉換後的結果，限定小數點後兩位。

數字猜謎遊戲

題目說明

實作一個數字猜謎遊戲，程式內部隨機產生一個 1 至 50 之間的秘密數字，讓使用者反覆猜測，並根據猜測結果給出提示：

  • 若猜測數字太小，提示「太小了！」
  • 若猜測數字太大，提示「太大了！」
  • 若猜中則印出「猜對了！」並結束遊戲

參考解答

import random

secret_number = random.randint(1, 50)  # 隨機選擇 1 到 50 之間的數字
print("歡迎來到數字猜謎遊戲！")
print("我已選好一個 1 到 50 之間的數字，請開始猜測。")

while True:
    try:
        guess = int(input("請輸入你的猜測: "))
    except ValueError:
        print("請輸入有效的整數！")
        continue

    if guess < secret_number:
        print("太小了！")
    elif guess > secret_number:
        print("太大了！")
    else:
        print("猜對了！恭喜你！")
        break

解說

程式使用 random.randint 產生秘密數字，並在 while 迴圈中反覆提示使用者進行猜測。根據條件判斷給出相應提示，直至使用者猜中為止，同時加入了基本的錯誤處理以避免非整數輸入。

文字統計分析器

題目說明

設計一個程式，提示使用者輸入一段文字，然後統計並輸出以下數據：

  • 總字數（包含所有字元）
  • 單詞數（依據空白分隔計算）
  • 非空白字元數（去除所有空白後的字元數）

參考解答

print("文字統計分析器")
text = input("請輸入一段文字: ")

# 總字數（包含所有字元）
total_chars = len(text)

# 單詞數（以空格分割）
words = text.split()
word_count = len(words)

# 非空白字元數（去除空白）
non_space_chars = len(text.replace(" ", ""))

print(f"總字數: {total_chars}")
print(f"單詞數: {word_count}")
print(f"非空白字元數: {non_space_chars}")

解說

程式先利用 len() 函式計算輸入文字的總字元數，再使用 split() 方法根據空白將文字分割成單詞來計算單詞數，最後利用 replace() 去除空白後計算非空白字元數。

ASCII 三角形

題目說明

撰寫一個程式，提示使用者輸入三角形的高度，然後利用星號 * 印出一個中心對齊的對稱三角形。每行星號數量為 2*i - 1，前置空格數量為 height - i。

參考解答

height = int(input("請輸入三角形的高度: "))

for i in range(1, height + 1):
    # 計算前置空格與星號數量
    spaces = " " * (height - i)
    stars = "*" * (2 * i - 1)
    print(spaces + stars)

解說

本程式先由使用者輸入三角形高度，接著利用 for 迴圈逐行計算每行所需的空格數與星號數，最終印出一個中心對齊的對稱三角形。

挑戰題 1：自定義乘法表範圍 🛈

請撰寫一個程式，讓使用者輸入乘法表的起始數字與結束數字，並選擇排列方式（正序或倒序）。根據輸入的範圍及排列方式，產生對應的乘法表。每一項格式為「a x b = c」，並以 tab 做間隔。

# 自定義乘法表範圍
while True:
    start = int(input("請輸入乘法表起始數字: "))
    end = int(input("請輸入乘法表結束數字: "))
    order = input("請選擇排列方式 (正序/倒序): ")
    
    if order == "正序":    
        if start < end:
            break
        else:
            print("起始/結束數字輸入錯誤，請重新輸入")
    elif order == "倒序":
        if end < start:
            break
        else:
            print("起始/結束數字輸入錯誤，請重新輸入")
    else:
        print("排列方式輸入錯誤，請重新輸入")

if order == "正序":
    nums = range(start, end + 1)
elif order == "倒序":
    nums = range(end, start - 1, -1)

for i in nums:
    for j in range(1, 10):
        print(f"{i}x{j}={i*j}", end="\t")
    print()  # 換行

挑戰題 2：多單位溫度轉換器 🛈

擴充原始溫度轉換器功能，讓使用者可以在攝氏、華氏和克爾文三種溫度單位間進行轉換。提供六個選項：攝氏轉華氏、攝氏轉克爾文、華氏轉攝氏、華氏轉克爾文、克爾文轉攝氏、克爾文轉華氏。根據使用者選擇提示輸入溫度，並輸出轉換結果。

def celsius_to_fahrenheit(c):
    return c * 9/5 + 32

def celsius_to_kelvin(c):
    return c + 273.15

def fahrenheit_to_celsius(f):
    return (f - 32) * 5/9

def fahrenheit_to_kelvin(f):
    return (f - 32) * 5/9 + 273.15

def kelvin_to_celsius(k):
    return k - 273.15

def kelvin_to_fahrenheit(k):
    return (k - 273.15) * 9/5 + 32

print("多單位溫度轉換器")
print("選擇轉換模式：")
print("1. 攝氏轉華氏")
print("2. 攝氏轉克爾文")
print("3. 華氏轉攝氏")
print("4. 華氏轉克爾文")
print("5. 克爾文轉攝氏")
print("6. 克爾文轉華氏")

choice = input("請輸入選項 (1-6): ")

if choice == "1":
    c = float(input("請輸入攝氏溫度: "))
    print(f"{c}°C 轉換為華氏溫度是 {celsius_to_fahrenheit(c):.2f}°F")
elif choice == "2":
    c = float(input("請輸入攝氏溫度: "))
    print(f"{c}°C 轉換為克爾文溫度是 {celsius_to_kelvin(c):.2f}K")
elif choice == "3":
    f = float(input("請輸入華氏溫度: "))
    print(f"{f}°F 轉換為攝氏溫度是 {fahrenheit_to_celsius(f):.2f}°C")
elif choice == "4":
    f = float(input("請輸入華氏溫度: "))
    print(f"{f}°F 轉換為克爾文溫度是 {fahrenheit_to_kelvin(f):.2f}K")
elif choice == "5":
    k = float(input("請輸入克爾文溫度: "))
    print(f"{k}K 轉換為攝氏溫度是 {kelvin_to_celsius(k):.2f}°C")
elif choice == "6":
    k = float(input("請輸入克爾文溫度: "))
    print(f"{k}K 轉換為華氏溫度是 {kelvin_to_fahrenheit(k):.2f}°F")
else:
    print("輸入錯誤，請重新執行程式。")

挑戰題 3：猜謎遊戲挑戰版 🛈

在原有數字猜謎遊戲基礎上，新增記錄使用者猜測次數的功能。當使用者猜中後，根據猜測次數給出不同評價：若少於 5 次，評價「太厲害了！」；介於 5 到 10 次，評價「表現不錯！」；超過 10 次，評價「還可以再加油！」

import random

secret_number = random.randint(1, 50)
print("挑戰版數字猜謎遊戲！")
print("我已選好一個 1 到 50 的數字，開始猜測吧！")

guess_count = 0

while True:
    try:
        guess = int(input("請輸入你的猜測: "))
    except ValueError:
        print("請輸入有效的整數！")
        continue

    guess_count += 1

    if guess < secret_number:
        print("太小了！")
    elif guess > secret_number:
        print("太大了！")
    else:
        print("猜對了！")
        break

print(f"你總共猜了 {guess_count} 次。")
if guess_count < 5:
    print("太厲害了！")
elif 5 <= guess_count <= 10:
    print("表現不錯！")
else:
    print("還可以再加油！")

挑戰題 4：進階文字統計分析器 🛈

在基本文字統計分析器的基礎上，除了計算總字數、單詞數與非空白字元數外，進一步統計每個字母（忽略大小寫）的出現頻率，並按字母順序列出結果。

print("進階文字統計分析器")
text = input("請輸入一段文字: ")

total_chars = len(text)
words = text.split()
word_count = len(words)
non_space_chars = len(text.replace(" ", ""))

print(f"總字數: {total_chars}")
print(f"單詞數: {word_count}")
print(f"非空白字元數: {non_space_chars}")

# 統計每個字母出現次數（忽略大小寫）
letter_counts = {}
for char in text.lower():
    if char.isalpha():
        letter_counts[char] = letter_counts.get(char, 0) + 1

print("字母出現頻率：")
for letter in sorted(letter_counts):
    print(f"{letter}: {letter_counts[letter]}")

挑戰題 5：多形狀 ASCII 圖形 🛈

設計一個程式，讓使用者選擇要印出的圖形形狀，包括正三角形、倒三角形和菱形。根據使用者輸入的高度，印出對應的 ASCII 圖形。菱形由正三角形和倒三角形（中間行不重複）組成。

print("多形狀 ASCII 圖形")
print("選擇圖形形狀：")
print("1. 正三角形")
print("2. 倒三角形")
print("3. 菱形")

choice = input("請輸入選項 (1-3): ")
height = int(input("請輸入圖形高度: "))

if choice == "1":
    # 正三角形
    for i in range(1, height + 1):
        spaces = " " * (height - i)
        stars = "*" * (2 * i - 1)
        print(spaces + stars)
elif choice == "2":
    # 倒三角形
    for i in range(height, 0, -1):
        spaces = " " * (height - i)
        stars = "*" * (2 * i - 1)
        print(spaces + stars)
elif choice == "3":
    # 菱形：先印正三角形，再印倒三角形（中間行不重複）
    for i in range(1, height + 1):
        spaces = " " * (height - i)
        stars = "*" * (2 * i - 1)
        print(spaces + stars)
    for i in range(height - 1, 0, -1):
        spaces = " " * (height - i)
        stars = "*" * (2 * i - 1)
        print(spaces + stars)
else:
    print("輸入錯誤！")
